# <a name="credential-messages-in-sharepoint-or-onedrive"></a>رسائل الاعتماد في SharePoint أو أندريف

<p>قد تتم دورياً يتم مطالبة المستخدمين بمكتب لبيانات الاعتماد إلى SharePoint على الإنترنت أو أندريف عند محاولة فتح المستندات.</p> <p>فيما يلي بعض الأمثلة عن رسائل الاعتماد:</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>تسجيل الدخول باستخدام الحساب الخاص بك التنظيمية</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>بيانات اعتماد مطلوبة</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>نحن غير قادرين على الاتصال الآن، تحقق من اتصال الشبكة.</p> <p>إذا كنت تواجه هذه الأعراض، الرجاء اتباع الخطوات الموجودة في <a href="https://support.microsoft.com/en-us/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint">هذه المقالة</a>.</p> <p>بدلاً من ذلك، إذا كنت تستخدم تسجيل الدخول الأحادي عليه (SSO) للمصادقة، الرجاء الرجوع إلى <a href="https://support.microsoft.com/en-us/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10">هذه المقالة</a> بدلاً من ذلك.</p> <p>&nbsp;</p>

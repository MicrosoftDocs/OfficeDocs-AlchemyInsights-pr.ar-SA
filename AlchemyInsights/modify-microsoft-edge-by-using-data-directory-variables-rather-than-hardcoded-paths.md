# <a name="modify-microsoft-edge-by-using-data-directory-variables-rather-than-hardcoded-paths"></a>تعديل Microsoft Edge باستخدام متغيرات دليل البيانات بدلا من مسارات مضمن

علي سبيل المثال ، في Windows ، لتخزين بيانات ملف التعريف ضمن بيانات التطبيق المحلي للمستخدم بدلا من الموقع الافتراضي ، عين نهج **أوسيرداتادير** إلى **$ {local_app_data} \edge\profile**. 

لمعرفه المزيد ، راجع [إنشاء متغيرات دليل بيانات مستخدمي Microsoft Edge](https://docs.microsoft.com/deployedge/edge-learnmore-create-user-directory-vars).

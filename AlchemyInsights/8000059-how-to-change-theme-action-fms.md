# <a name="how-to-change-your-theme"></a>كيفية تغيير الموضوع الخاص بك

لتغيير النسق الخاص بك في Outlook على الويب أو Outlook.com، انتقل إلى **إعدادات** واختيار سمة ضمن **سمة**. لمشاهدة كافة السمات المتوفرة، حدد **عرض كل** **موضوع**.

# <a name="welcome-to-the-microsoft-365-lighthouse-private-preview"></a>مرحبا بك في Microsoft 365 Lighthouse Private Preview

أنت تشارك في المعاينة الخاصة لمناارة Microsoft 365. إذا كنت ترغب في تقديم ملاحظات حول Microsoft 365 Lighthouse، فانشر التفاصيل إلى موقع [User Voice.](https://aka.ms/M365Lighthouseuservoice) إذا كان لديك سؤال أو تواجه مشكلة أثناء استخدام Microsoft 365 Lighthouse، فيرجى إنشاء طلب دعم للحصول على مزيد من المساعدة.

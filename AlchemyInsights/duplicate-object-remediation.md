# <a name="duplicate-object-synchronization-errors"></a>أخطاء مزامنة الكائنات المكررة

إذا تلقيت أحد الأخطاء التالية أثناء المزامنة:

- يوجد كائن متزامن بنفس عنوان الوكيل بالفعل في دليل خدمات Microsoft عبر الإنترنت.

- رسالة مشابهة لاسم UserPrincipalName.

استخدم تشخيص AADConnect Health الجديد لحل أخطاء الكائنات المكررة. لمزيد من المعلومات، راجع [تشخيص أخطاء مزامنة السمات المكررة وإصلاحها](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).

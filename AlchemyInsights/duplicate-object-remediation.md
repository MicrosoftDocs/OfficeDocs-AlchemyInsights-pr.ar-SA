# <a name="duplicate-object-synchronization-errors"></a>تكرار أخطاء مزامنة الكائن

إذا تلقيت أحد الأخطاء التالية أثناء مزامنة:

- يوجد بالفعل كائن متزامنة بنفس عنوان الوكيل في دليل "خدمات Microsoft عبر إنترنت".

- رسالة مشابهة ل UserPrincipalName.

استخدام التشخيص الصحي آدكوننيكت جديدة لحل أخطاء كائن مكرر. لمزيد من المعلومات، راجع [تشخيص وإصﻻح أخطاء المزامنة السمة المكررة](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).

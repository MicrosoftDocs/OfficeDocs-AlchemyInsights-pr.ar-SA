# <a name="duplicate-object-synchronization-errors"></a>أخطاء مزامنة كائن مكرره

إذا ظهرت أحدي الأخطاء التالية اثناء المزامنة:

- يوجد كائن متزامن بنفس عنوان الوكيل بالفعل في دليل Microsoft Online Services.

- رسالة مماثله لكل منها.

استخدم تشخيص حماية AADConnect الجديد لحل أخطاء الكائن المكررة. لمزيد من المعلومات ، راجع [تشخيص أخطاء مزامنة السمات المكررة وريميدياتيها](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).

# <a name="one-of-your-on-premises-federation-service-certificates-is-expiring"></a>انتهت صلاحيه أحدي شهادات الخدمة الموحدة المحلية لديك

لحل هذه المشكلة ، اتبع الخطوات التالية:
  
- قم بتثبيت الوحدة النمطية ل Microsoft Azure Active directory ل Windows PowerShell علي الكمبيوتر (إذا لم تكن الوحدة النمطية مثبته بالفعل). للقيام بذلك ، انتقل إلى [Azure Active Directory PowerShell ل Graph ](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)
    
- اتبع الخطوات الواردة في القسم "السيناريو 1: انتهت صلاحيه الرمز المميز للإعلان عن شهادة" في [الحالة "هناك مشكله في الوصول إلى الموقع" من AD FS عندما يقوم مستخدم خارجي بتسجيل الدخول إلى Microsoft 365 أو Azure أو Intune](https://support.microsoft.com/help/2713898/there-was-a-problem-accessing-the-site-error-from-ad-fs-when-a-federat).
    
- اتبع الخطوات الواردة في [كيفيه تحديث إعدادات مجال متحد أو إصلاحها في Microsoft 365 أو Azure أو Intune](https://support.microsoft.com/help/2647048/how-to-update-or-repair-the-settings-of-a-federated-domain-in-office-3).
    
للحصول علي مزيد من المعلومات حول تجديد شهادات الاتحاد ، راجع [تجديد الشهادة ل O365 و AZURE AD](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-o365-certs).

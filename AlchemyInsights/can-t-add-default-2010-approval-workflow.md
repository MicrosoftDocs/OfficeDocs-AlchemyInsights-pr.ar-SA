# <a name="cant-add-default-2010-approval-workflow"></a>لا يمكن إضافة الافتراضي 2010 "سير عمل الموافقة"

في مجموعة موقع Microsoft SharePoint، لا يمكنك إضافة سير عمل القابل لإعادة الاستخدام على الصعيد العالمي (مثل "الموافقة-SharePoint 2010") إلى قائمة أو مكتبة. قد يكون سير العمل مفقودة. لحل هذه المشكلة راجع [يتعذر إضافة سير عمل القابل لإعادة الاستخدام على الصعيد العالمي من قائمة SharePoint أو مكتبته](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

# <a name="solutions-for-error-30088-4"></a>حلول للخطأ 30088-4

حاول هذه الحلول لإصلاح الخطأ [30088-4:](https://support.office.com/article/d5df89a9-0507-4b4c-92f9-22f457e630aa?wt.mc_id=Alchemy_ClientDIA)
  
- إعادة تشغيل جهاز الكمبيوتر الخاص بك وحاول [تثبيت Office](https://portal.office.com/OLS/MySoftware.aspx).

- قم بتنزيل وتثبيت [مساعد الإصلاح والدعم](https://aka.ms/SARA-OfficeUninstall-Alchemy).

1. عند فتحه، من **منتجات إلغاء تثبيت Office** حدد الإصدار الذي تريد إلغاء تثبيته وحدد **التالي**.

2. استمر في العمل عبر الشاشات وقم بإعادة تشغيل الكمبيوتر عندما تتم مطالبتك بذلك.

    بعد إعادة التشغيل، يتم إعادة فتح "مساعد الدعم والاسترداد" تلقائيًا لإكمال عملية إلغاء التثبيت.

3. اتبع المطالبات المتبقية، [وإعادة تثبيت Office](https://portal.office.com/OLS/MySoftware.aspx) أو إغلاق المساعد.

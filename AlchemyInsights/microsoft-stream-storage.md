# <a name="microsoft-stream-storage"></a>مساحة تخزين Microsoft Stream

تتلقي Microsoft Stream مستاجرا تخصيصا ثابتا لمساحة التخزين التي تبلغ 500 غيغابايت ومساحة تخزين اضافيه تبلغ 0.5 غيغابايت لكل مستخدم مرخص.
للحصول علي مزيد من التفاصيل حول مساحة التخزين ، راجع:

- [نظره عامه حول تخزين Microsoft Stream](https://docs.microsoft.com/stream/license-overview#storage)
- [أضافه مساحة تخزين إلى Microsoft Stream](https://docs.microsoft.com/stream/storage-add-on)
- [الحصص النسبية والقيود في Microsoft Stream](https://docs.microsoft.com/stream/quotas-and-limitations)

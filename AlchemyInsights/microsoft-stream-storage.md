# <a name="microsoft-stream-storage"></a>تخزين تيار مايكروسوفت

يحصل مستأجرو Microsoft Stream على تخصيص ثابت بسعة تخزين ية تبلغ 500 غيغابايت ومساحة تخزين إضافية قدرها 0.5 غيغابايت لكل مستخدم مرخص.
لمزيد من التفاصيل حول التخزين، راجع:

- [نظرة عامة على تخزين Microsoft Stream](https://docs.microsoft.com/stream/license-overview#storage)
- [إضافة سعة تخزين إلى Microsoft Stream](https://docs.microsoft.com/stream/storage-add-on)
- [الحصص والقيود في Microsoft Stream](https://docs.microsoft.com/stream/quotas-and-limitations)

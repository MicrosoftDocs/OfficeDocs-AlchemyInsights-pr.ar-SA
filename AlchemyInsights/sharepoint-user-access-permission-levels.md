# <a name="user-permissions-in-sharepoint"></a>أذونات المستخدم في SharePoint

بعد إنشاء موقع SharePoint، قد تحتاج إلى توفير أو تقييد وصول المستخدم إلى الموقع أو محتوياته. أسهل طريقة للعمل مع أذونات لاستخدام مستويات [المجموعات الافتراضية](https://support.office.com/article/default-sharepoint-groups-13bb2b6b-dd8c-447e-b71b-0e4bb9efe1d3) والأذونات المتوفرة، التي تغطي السيناريوهات الأكثر شيوعاً. ولكن، إذا أردت ذلك، يمكنك تعيين أذونات موضوعة بدقة أكثر ما يتجاوز المستويات الافتراضية. لمزيد من المعلومات، راجع [فهم "مستويات الإذن" في SharePoint](https://docs.microsoft.com/sharepoint/understanding-permission-levels).

لمزيد من المعلومات حول إدارة مستويات أذونات معينة، الرجاء زيارة الارتباطات الموجودة أدناه:

- [تحرير وإدارة الأذونات لقائمة SharePoint أو مكتبته](https://support.office.com/article/customize-permissions-for-a-sharepoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782)

- [إنشاء وإدارة مجموعات SharePoint](https://support.office.com/article/create-and-manage-sharepoint-groups-b1e3cd23-1a78-4264-9284-87fed7282048)

- [كيفية إنشاء وتحرير "مستويات الأذونات"](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)

- [إعداد وإدارة طلبات الوصول](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3)

- [تغيير الأذونات على مجلد فرعي](https://support.office.com/article/change-the-permissions-on-a-subfolder-5427bd7c-f20a-4f75-8cf2-5359dd45a1a6)

- [مشاركة الملفات SharePoint أو المجلدات الموجودة في Office 365](https://support.office.com/article/share-sharepoint-files-or-folders-1fe37332-0f9a-4719-970e-d2578da4941c)

- [مشاركة موقع](https://support.office.com/article/share-a-site-958771a8-d041-4eb8-b51c-afea2eae3658)

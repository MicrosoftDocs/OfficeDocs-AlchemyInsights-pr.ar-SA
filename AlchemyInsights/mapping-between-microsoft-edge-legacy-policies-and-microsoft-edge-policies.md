# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>تعرف على التعيين بين سياسات Microsoft Edge القديمة ونهج Microsoft Edge

يتم تعيين "سياسات المجموعة" ل Microsoft Edge Legacy (الإصدارات 45 والإصدارات السابقة) وإعدادات إدارة أجهزة المحمول (MDM) إلى سياسات Microsoft Edge ذات الصلة، الإصدار 80. لمزيد من المعلومات، راجع [تعيين نهج Microsoft Edge القديم ل Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge).

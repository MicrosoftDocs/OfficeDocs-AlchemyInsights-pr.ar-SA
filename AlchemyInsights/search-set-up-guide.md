# <a name="get-started-with-microsoft-search-using-the-set-up-guide"></a>بدء استخدام Microsoft Search باستخدام دليل إعداد

يوفر Microsoft Search واجهة سهلة الاستخدام للبحث عن النتائج والعثور عليها عبر Microsoft 365 وتطبيقاتها في مؤسستك. يتم تضمين Microsoft Search مع اشتراكك المؤهل Microsoft 365 دون أي تكلفة إضافية. 

استخدم دليل [إعداد البحث من Microsoft](https://go.microsoft.com/fwlink/?linkid=2156919) لتكوين ما إذا كنت تريد بدء البحث مع مجموعة من المستخدمين أو طرحه للجميع في المؤسسة. ستعين المسؤولين عن البحث ومحرري البحث وخصص تجربة البحث للمستخدمين باستخدام الإجابات والخيارات الإضافية، مثل إضافة ملحق Bing إلى Chrome أو تعيين Bing كمحرك بحث افتراضي.

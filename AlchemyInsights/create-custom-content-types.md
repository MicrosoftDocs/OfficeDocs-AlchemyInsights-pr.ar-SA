# <a name="create-custom-content-types"></a>إنشاء أنواع محتوى مخصصة

1. حدد القائمة التي تريد العمل معها على التشغيل السريع، أو انقر فوق **الإعدادات** (رمز الترس) ثم انقر فوق **إعدادات الموقع**.
2. انقر فوق أنواع **محتوى الموقع** ضمن معارض **مصمم الويب**.
3. في صفحة أنواع محتوى الموقع، انقر فوق إنشاء.
4. في صفحة نوع محتوى الموقع الجديد، قم بتعبئة المعلومات لنوع المحتوى المخصص.

لمزيد من المعلومات، راجع [أنواع المحتوى المخصصة](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991).

# <a name="dashboard-not-loading"></a>لوحة المعلومات لا يتم تحميلها

**اعراض:**

قد تكون لوحة المعلومات عالقة مع الدوار التحميل، أو قد تظهر رسالة خطأ عامة: "حدث خطأ ما. تحقق من التفاصيل الفنية لمزيد من التفاصيل".

**دليل استكشاف الأخطاء وإصلاحها:**

يرجى الاطلاع على [لوحة معلومات UCI لا يتم تحميلها بعد إصدار 2003.5.](https://support.microsoft.com/help/4558635/uci-dashboard-not-loading-after-the-2003-5-release)

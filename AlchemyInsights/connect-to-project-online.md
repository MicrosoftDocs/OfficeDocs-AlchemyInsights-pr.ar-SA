# <a name="connect-to-project-online"></a>الاتصال بالمشروع عبر الإنترنت

يمكنك الاتصال بـ Project Online باستخدام:

- عميل Project Online لسطح المكتب للاتصال بـ Project Online في بيئة Office 365. لمزيد من المعلومات، راجع [الاتصال بالمشروع عبر الإنترنت مع عميل سطح المكتب المشروع عبر إنترنت](https://docs.microsoft.com/projectonline/connect-to-project-online-with-the-project-online-desktop-client).  
- خدمة Power BI لإلغاء تأمين المعارف من Project Web App للمساعدة في إدارة المشاريع والمحافظ والموارد. لمزيد من المعلومات، راجع [الاتصال بتطبيق Project Web باستخدام Power BI](https://docs.microsoft.com/power-bi/service-connect-to-project-online).

# <a name="connect-to-project-online"></a>الاتصال ب Project Online

يمكنك الاتصال ب Project Online باستخدام:

- عميل سطح مكتب Project Online للاتصال ب Project Online في بيئة Office 365. لمزيد من المعلومات ، راجع [الاتصال ب Project online باستخدام عميل سطح مكتب Project online](https://docs.microsoft.com/projectonline/connect-to-project-online-with-the-project-online-desktop-client).  
- خدمه power BI للغاء تامين الرؤية الدقيقة من Project Web App للمساعدة علي أداره المشاريع والقوائم والموارد. لمزيد من المعلومات ، راجع [الاتصال ب Project Web App باستخدام POWER BI](https://docs.microsoft.com/power-bi/service-connect-to-project-online).

# <a name="cant-select-contacts-when-composing-email"></a>لا يمكن تحديد جهات الاتصال عند إنشاء بريد إلكتروني

عند إنشاء رسالة بريد إلكتروني جديدة، لا أرى بعض العملاء قائمة جهات الاتصال عند النقر فوق الزر **إلى** أو **نسخة** . لتحديد جهة اتصال، ابدأ الكتابة في **إلى** أو **نسخة** الحقول وسيتم عرض جهات الاتصال ذات الصلة.

# <a name="cant-select-contacts-when-composing-email"></a>لا يمكن تحديد جهات الاتصال عند إنشاء البريد الإلكتروني

عند إنشاء بريد إلكتروني جديد، لا يرى بعض العملاء قائمة جهات الاتصال الخاصة بهم عند النقر فوق **الزرين "إلى"** أو **"نسخة".** لتحديد جهة اتصال، ابدأ الكتابة في حقول **"إلى"** أو **"نسخة"** وسيتم عرض جهات الاتصال ذات الصلة.

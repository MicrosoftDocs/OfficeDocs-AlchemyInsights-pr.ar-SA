# <a name="change-your-profile-name-in-outlookcom"></a>تغيير اسم التشكيل الجانبي الخاص بك في Outlook.com

1. انتقل إلى [معلومات الحساب الخاص بك ل Microsoft](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. حدد **تحرير اسم**.
3. أدخل الاسم الأول واسم العائلة، وحدد **حفظ**.

الرجاء الانتظار 24 ساعة لاسم ملف التعريف الخاص بك تفعيل التغييرات.

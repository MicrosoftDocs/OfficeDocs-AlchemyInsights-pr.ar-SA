# <a name="change-your-profile-name-in-outlookcom"></a>تغيير اسم ملف التعريف الخاص بك في Outlook.com

1. انتقل إلى [معلومات حساب Microsoft الخاص بك](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. حدد **تحرير الاسم**.
3. ادخل الاسم الأول واسم العائلة ، وحدد **حفظ**.

الرجاء السماح بما يصل إلى 24 ساعة لتغيير اسم ملفك الشخصي ليدخل حيز التنفيذ.

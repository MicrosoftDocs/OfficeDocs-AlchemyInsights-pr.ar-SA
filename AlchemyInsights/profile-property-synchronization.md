# <a name="profile-property-synchronization"></a>مزامنة خصائص ملف التعريف

من غير الممكن تعيين خصائص اضافيه لاستيراد AD للمزامنة من Azure Active Directory إلى تطبيق ملف تعريف المستخدم. استيراد AD يقتصر علي مجموعه من الخصائص التي تم تكوينها مسبقا لضمان الأداء المتناسق لمهمة المؤقت.
  
[مزيد من المعلومات حول خصائص ملف التعريف التي تتم مزامنتها](https://go.microsoft.com/fwlink/?linkid=875671)

# <a name="profile-property-synchronization"></a>مزامنة خاصيه ملف التعريف

لا يمكن تعيين الخصائص الاضافيه لاستيراد AD للمزامنة من Azure Active directory إلى تطبيق ملف تعريف المستخدم. يقتصر استيراد AD علي مجموعه من الخصائص المكونة مسبقا لضمان الأداء المتناسق لمهمة المؤقت.
  
[تعرف علي المزيد حول خصائص ملف التعريف التي تتم مزامنتها](https://go.microsoft.com/fwlink/?linkid=875671)

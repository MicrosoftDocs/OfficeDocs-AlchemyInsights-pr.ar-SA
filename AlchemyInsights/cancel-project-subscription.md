# <a name="cancel-project-subscription"></a>إلغاء اشتراك المشروع

يمكن للمسؤولين إلغاء اشتراك المشروع لمؤسساتهم من **مركز إدارة Microsoft 365**. 

1. حدد خيار **الفوترة.**

2. انقر على [المنتجات & الخدمات](https://admin.microsoft.com/AdminPortal/Home?adminportal=1&msCV=%2BbOQtMNsz0ei8f5z.0.36#/subscriptions) لعرض كافة الاشتراكات لمؤسستك.

3. تحديد موقع خطة المشروع لإلغاء.

4. انقر فوق **إلغاء الاشتراك**.

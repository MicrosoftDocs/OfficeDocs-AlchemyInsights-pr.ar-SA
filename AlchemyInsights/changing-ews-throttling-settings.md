# <a name="changing-ews-throttling-settings"></a>تغيير إعدادات التحكم في EWS

يرجى تشغيل الاختبار التلقائي الذي سيسمح لك بتعديل نهج التحكم EWS لمده الترحيل.

يرجى ملاحظه ان تغييرات نهج كبح EWS لا تؤثر علي أنواع الترحيل التالية (باستخدام أدوات Microsoft): مختلط أو كلي/مجهز (RPC/HTTP) أو IMAP أو G Suite أو خدمه استيراد المجلد العام أو PST.

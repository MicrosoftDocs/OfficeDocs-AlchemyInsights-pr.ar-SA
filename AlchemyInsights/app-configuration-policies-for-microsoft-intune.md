# <a name="app-configuration-policies-for-microsoft-intune"></a>نُهج تكوين التطبيق لـ Microsoft Intune

توفر نُهج تكوين التطبيق في Microsoft Intune إعدادات التكوين لتطبيقات iOS أو Android. تسمح إعدادات التكوين هذه بتخصيص التطبيق. دعم نهج التكوين محدود بواسطة إصدارات نظام التشغيل والتطبيقات.

لمزيد من التفاصيل، راجع [نُهج تكوين التطبيقات لـ Microsoft Intune](https://docs.microsoft.com/intune/app-configuration-policies-overview).

# <a name="cant-download-files-in-chrome"></a>لا يمكن تنزيل الملفات في Chrome

**العدد** نحن على علم بمشكلة تمنع بعض العملاء من تنزيل الملفات عند استخدام متصفح Google Chrome. 

**الحل البديل** استخدم [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) أو مستعرض ويب آخر.
تعرف على المزيد حول كيفية [تنزيل الملفات والمرفقات Outlook.com](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).

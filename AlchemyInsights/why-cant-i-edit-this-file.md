# <a name="why-cant-i-edit-this-file"></a>لماذا لا يمكنني تحرير هذا الملف؟

إذا تم تأمين مستند أو ورقة عمل أو عرض تقديمي للتحرير ولا يمكن تحريره في تطبيقات سطح مكتب Windows، فقد يتم تحرير الملف من قبل شخص آخر، أو تم سحب الملف منه.

إلقاء نظرة سريعة على بعض الأسباب [التي قد لا تتمكن من تحرير الملف](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).

# <a name="join-a-team-by-code"></a>الانضمام إلى فريق باستخدام التعليمات البرمجية

تدرك Microsoft مشكلة عدم تمكن المستخدمين من  إنشاء رمز الانضمام إلى فريق أو الانضمام إلى فريق باستخدام ميزة "الانضمام حسب التعليمات **البرمجية".** كحل بديل، يمكن لمالكي الفريق إضافة مستخدمين يدويا، حسب الحاجة.

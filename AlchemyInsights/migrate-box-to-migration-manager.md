# <a name="migrate-box-to-microsoft-365-with-migration-manager"></a>مربع الترحيل إلى Microsoft 365 باستخدام إدارة الترحيل

تعاون في العمل في مكان واحد من خلال نقل مستندات Box والبيانات والمستخدمين إلى OneDrive و SharePoint و Teams في Microsoft 365.

لمزيد من المعلومات، راجع [مربع الترحيل إلى Microsoft 365 باستخدام إدارة الترحيل](https://docs.microsoft.com/sharepointmigration/mm-box-overview).

# <a name="install-and-configure-onedrive"></a>تثبيت وتكوين أونيدريفي

عميل المزامنة أندريف المثبتة مع Windows 10. لتثبيت أندريف لنظام التشغيل Windows 7 أو Mac، أو للحصول على التطبيق المحمول للهاتف أو الكمبيوتر اللوحي الخاص بك، انتقل إلى [http://onedrive.com/download](http://onedrive.com/download).
  
لإعداد عميل المزامنة أندريف في Windows، راجع:
  
- [إعداد الكمبيوتر لمزامنة جهاز أندريف لملفات العمل في Office 365](https://go.microsoft.com/fwlink/?linkid=533375)
    
- [مزامنة الملفات SharePoint باستخدام عميل المزامنة أونيدريفي الجديدة](https://go.microsoft.com/fwlink/?linkid=871666)

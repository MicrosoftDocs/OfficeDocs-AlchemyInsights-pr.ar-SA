# <a name="read-receipts"></a>إعلام بالقراءة

لا يمكنك طلب إعلام بقراءة في Outlook.com، ولكن يمكنك طلب إيصالات بالقراءة للرسائل Outlook.com التي تم إرسالها باستخدام Outlook ل Windows. لمعرفة كيفية إجراء ذلك، راجع [إضافة وطلب قراءة الإيصالات وتسليم رسائل الإعلام](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141?wt.mc_id=Office_Outlook_com_Alchemy).
  
لاختيار كيفية Outlook.com استجابة لطلبات إيصال بالقراءة، انتقل إلى [إعدادات إيصال القراءة](https://outlook.live.com/mail/options/mail/handling/readReceipts).

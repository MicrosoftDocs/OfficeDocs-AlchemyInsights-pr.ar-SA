# <a name="map-a-sharepoint-library-to-a-network-drive"></a>تعيين مكتبة SharePoint إلى محرك أقراص شبكة الاتصال

للعمل مع مكتبات الموقع في "مستكشف الملفات"، نوصي باستخدام عميل المزامنة أندريف بدلاً من تعيين المكتبات إلى محركات أقراص الشبكة. تعيين محرك الأقراص مجرد حل مؤقت. استخدام عميل المزامنة يجعل الملفات متوفرة للمستخدمين حتى عندما كنت غير متصل. عميل المزامنة أيضا يوفر أداء أفضل ويسمح بالوصول إلى الملفات دون الحاجة إلى تحميل هذه [الملفات عند](https://support.office.com/article/Learn-about-OneDrive-Files-On-Demand-0E6860D3-D9F3-4971-B321-7092438FB38E) استخدام ولا يتطلب برنامج الإعداد باستخدام برنامج Internet Explorer. 
  
- [مزامنة الملفات باستخدام عميل المزامنة أندريف جديدة في Windows](https://go.microsoft.com/fwlink/?linkid=866427)
    
- [طرق العمل مع موقع مكتبة ملفات في "مستكشف الملفات"](https://go.microsoft.com/fwlink/?linkid=866291)
    
- [تكوين واستكشاف محركات أقراص الشبكة المعينة](https://support.microsoft.com/kb/2616712)

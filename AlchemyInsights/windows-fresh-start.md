# <a name="windows-fresh-start"></a>بدء التشغيل الجديد لـ Windows

يوفر إجراء جهاز Windows Fresh Start بديلاً للإجراءات Device Wipe و Device Retire مع خيار الاحتفاظ بحالة تسجيل الجهاز مع تطبيقات وإعدادات المجلدات الرئيسية للمستخدم.

لمزيد من المعلومات، راجع [استخدام "البدء الجديد" لإعادة تعيين أجهزة Windows 10 باستخدام Intune](https://docs.microsoft.com/intune/device-fresh-start).

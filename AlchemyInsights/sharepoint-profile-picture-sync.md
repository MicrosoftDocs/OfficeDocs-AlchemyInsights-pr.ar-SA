# <a name="when-do-my-profile-changes-sync-to-the-sharepoint-user-profile-application"></a>متى تتم مزامنة ملف التعريف الخاص بي مع تطبيق ملف تعريف المستخدم SharePoint؟

تتطلب مزامنة صورة الملف الشخصي في SharePoint Online إجراء ً من المستخدم لبدء مزامنة صورته. لا توجد مزامنة تلقائية للصور إلى SharePoint Online. SharePoint Online مزامنة الصور فقط من Exchange Online، وليس Azure Active Directory.

إذا لم يقم مستخدم بإجراء من شأنه أن يطالب بالمزامنة أو قام بتحديث صورة ملفه الشخصي ولم يظهر المستخدم الجديد في SharePoint Online، فقد يحتاج المستخدم إلى المطالبة بمزامنته. وإليك كيف يمكنهم القيام بذلك:

1. افتح متصفح ويب في جلسة عمل In-Private أو التصفح المتخفي [واذهب إلى Office.com](https://www.office.com/).

2. قم بتسجيل الدخول باستخدام حساب العمل أو المدرسة، وانقر على لوحة OneDrive أو SharePoint.

3. [إضافة صورة ملفهم الشخصي إلى Microsoft 365](https://support.office.com/article/Add-your-profile-photo-to-Office-365-2eaf93fd-b3f1-43b9-9cdc-bdcd548435b7).

[معلومات حول مزامنة صورة الملف الشخصي في Microsoft 365](https://support.office.com/article/Information-about-user-profile-synchronization-in-SharePoint-Online-177eb196-5887-43c9-84c3-b98a43d35129)

# <a name="adding-a-mailbox-to-a-user"></a>إضافة صندوق بريد إلى مستخدم

لإضافة علبة بريد إلى مستخدم تم إنشاؤها مسبقاً أو مستخدم جديد، يجب تعيين المستخدم ترخيص لـ Exchange عبر إنترنت. إذا كان لدى المستخدم تراخيص معينة بالفعل، حدد Exchange عبر الإنترنت ضمن قسم التطبيقات في صفحة الترخيص لبدء العملية لإنشاء علبة بريد للمستخدم. لمزيد من المعلومات، راجع [تعيين التراخيص للمستخدمين](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users).

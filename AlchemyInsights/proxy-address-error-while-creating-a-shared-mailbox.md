# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>خطأ في عنوان الوكيل أثناء إنشاء علبة بريد مشتركة

إذا حاولت إنشاء علبة بريد مشتركة وتلقيت هذا الخطأ "يتم بالفعل استخدام عنوان الوكيل "SMTP:alias@domain.com..."، يتم بالفعل أخذ عنوان البريد الإلكتروني الذي اخترته بواسطة علبة بريد أخرى في مؤسستك.
  
تحتاج إلى العثور على المستخدم أو المجموعة أو صندوق البريد المشترك الذي يحتوي على عنوان البريد الإلكتروني هذا وحذفه أو تغيير عنوان البريد الإلكتروني. ثم يمكنك إنشاء صندوق بريد مشترك مع عنوان البريد الإلكتروني المحرر. استخدم البحث على الصفحة الرئيسية للعثور عليه.
  
إذا كنت لا تريد حذف عنوان البريد الإلكتروني الحالي، فاختر عنوان بريد إلكتروني جديد لصندوق البريد المشترك.

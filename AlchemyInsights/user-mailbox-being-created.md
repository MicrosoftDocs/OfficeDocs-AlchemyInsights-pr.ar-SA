# <a name="your-users-mailbox-is-being-created"></a>يتم إنشاء علبه بريد المستخدم الخاصة بك

قد يستغرق استخدام علبه بريد جديده مده تصل إلى ساعة.
  
إذا تم "تحضير علبه بريد المستخدم" لأكثر من ساعة ، فيمكنك محاولة حذف المستخدم وأعاده إنشائه أو استخدام الخيار "دعنا نتصل بك" للتحدث إلى أحد مهندسي الدعم.

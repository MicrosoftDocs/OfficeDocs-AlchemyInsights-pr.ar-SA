# <a name="your-users-mailbox-is-being-created"></a>يتم الآن إنشاء صندوق بريد المستخدم الخاص بك

يمكن أن يستغرق ساعة قبل صندوق بريد جديد جاهز للاستخدام.
  
إذا علبة البريد الخاصة مستخدم "إعداد" لأكثر من ساعة، يمكنك محاولة حذف وإعادة إنشاء المستخدم ثم أو استخدام الخيار "أدعوك" لإجراء محادثات مع مهندس دعم.

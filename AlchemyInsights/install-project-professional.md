# <a name="install-project-professional"></a>تثبيت Project Professional

راجع [تثبيت المشروع](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281?ui=en-US&rs=en-US&ad=US) للتعرف على كيفية تثبيت إصدار الاشتراك "المشروع على الإنترنت"، وإصدار الاشتراك غير المشروع عام 2019، 2016 أو 2013. 

مشروع 2010 أو 2007، اتبع الخطوات الموجودة في [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) أو [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

بيع المشروع كتطبيق مستقل، حتى إذا لم يتم ذلك حتى الآن، و [شراء أو حاول المشروع](https://products.office.com/project).

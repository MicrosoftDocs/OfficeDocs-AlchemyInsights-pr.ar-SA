# <a name="proxyaddress-incorrect"></a>عنوان الوكيل غير صحيح

عند مزامنة كائن إلى Azure AD، تتم مقارنة القيم المحددة في سمة proxyAddresses في "الدليل النشط" بقواعد Azure AD، ثم يتم ملء سمة proxyAddresses في Azure AD. لذلك، قد لا تكون قيم سمة proxyAddresses للكائن في "الدليل النشط" نفس قيم سمة proxyAddresses في Azure AD.
  
لمعرفة المزيد حول كيفية ملء عنوان الوكيل، راجع [كيفية ملء سمة proxyAddress في Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).

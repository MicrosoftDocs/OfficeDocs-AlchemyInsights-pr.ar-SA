# <a name="setting-up-a-hybrid-environment"></a>اعداد بيئة مختلطة

للسماح للمستخدمين بالعمل معا بشكل سلس بغض النظر عن البيانات المخزنة علي خوادم محليه أو في Microsoft 365 ، قد تحتاج إلى اعداد بيئة مختلطة. للحصول علي نظره عامه حول خياراتك ، راجع [تكامل Microsoft 365 مع البيئات المحلية](https://docs.microsoft.com/office365/enterprise/office-365-integration).

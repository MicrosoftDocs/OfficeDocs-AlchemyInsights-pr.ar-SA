# <a name="set-up-a-microsoft-365-education-subscription"></a>إعداد اشتراك Microsoft 365 Education

للحصول على الإرشادات، الرجاء مشاهدة [فيديو صوتي وصفي](https://aka.ms/M365EduSetup) لعملية إعداد Education بالكامل.

**مزيد من الموارد المفيدة**

- راجع [إنشاء حساب مستأجر Office 365 للحصول](https://docs.microsoft.com/microsoft-365/education/deploy/create-your-office-365-tenant) على نظرة عامة.
- راجع [بدء العمل باستخدام Microsoft 365 Education](https://docs.microsoft.com/education/) لمسؤولي تقنيات المعلومات والمطورين والشركاء.

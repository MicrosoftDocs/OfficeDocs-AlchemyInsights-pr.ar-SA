# <a name="remove-a-hub-site"></a>إزالة موقع لوحة الوصل

إلغاء تسجيل موقع لوحة وصل في مركز مسؤول SharePoint جديدة
  
1. في مركز مسؤول SharePoint جديدة ضمن **مواقع**، انقر فوق **مواقع نشطة**. 
    
2. حدد الموقع وانقر فوق **موقع لوحة الوصل**ثم انقر فوق **إلغاء التسجيل كموقع لوحة الوصل**. 
    
3. انقر فوق **"موافق"**.

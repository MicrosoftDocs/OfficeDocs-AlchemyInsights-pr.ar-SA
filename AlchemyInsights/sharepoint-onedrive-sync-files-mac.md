# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>تثبيت ملفات OneDrive أو SharePoint وإعدادها ومزامنتها على Mac 

تثبيت وإعداد ومزامنة ملفات OneDrive أو SharePoint على Mac مع الإرشادات من [ملفات المزامنة مع عميل مزامنة OneDrive على Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

إذا تم تثبيت تطبيق OneDrive والإعداد على جهاز Mac، ولكن المستخدمين يواجهون مشكلات في المزامنة، فراجع [مشكلات مزامنة الإصلاح مع OneDrive (Mac).](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756)

> [!NOTE]
> لا تتوفر ميزة نقل المجلد المعروف في مزامنة OneDrive لنظام التشغيل Windows لنظام التشغيل Mac.

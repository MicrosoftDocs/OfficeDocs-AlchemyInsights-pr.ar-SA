# <a name="bulk-delete-user"></a>مستخدم الحذف المجمع

يمكنك أزاله عدد كبير من المستخدمين من المجموعة باستخدام ملف قيم مفصوله بفاصله (CSV) لحذف المستخدمين بشكل مجمع. لحذف مستخدمين مجمعين باستخدام Azure AD portal ، راجع [المستخدمون الذين لديهم حذف مجمع في Azure Active directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).

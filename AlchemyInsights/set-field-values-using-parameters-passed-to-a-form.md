# <a name="set-field-values-using-parameters-passed-to-a-form---lookup-fields"></a>تعيين قيم الحقل باستخدام معلمات تم تمريرها إلى نموذج - حقول بحث

تعيين القيمة لحقول البحث ومثال: استخدام Xrm.Utility.openEntityForm لفتح إطار جديد. لا يمكنك تعيين قيم لـ partylist أو فيما يتعلق عمليات البحث.

لمزيد من المعلومات، راجع [تعيين قيم حقول البحث](https://docs.microsoft.com/previous-versions/dynamicscrm-2016/developers-guide/gg334375(v=crm.8)#set-values-for-lookup-fields).

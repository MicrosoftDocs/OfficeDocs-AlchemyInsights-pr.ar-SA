# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>استخدم سارة لتشخيص المشكلات وحلها Outlook مصادقة

نوصي بشدة استخدام [يحتفظ Outlook طلب كلمة المرور الخاصة بي](https://aka.ms/SaRA-OutlookPwdPrompt) التشخيص على جهاز متأثر لاستكشاف المشكلات وإصلاحها حيث يطالب Outlook باستمرار بكلمة مرور. يقوم بعمليات الفحص المؤتمت هذا [سارة](https://diagnostics.office.com/#/) التشخيص وإرجاع الحلول الممكنة لتتمكن من استخدامها لمواجهة أية مشكلات تم اكتشافها.

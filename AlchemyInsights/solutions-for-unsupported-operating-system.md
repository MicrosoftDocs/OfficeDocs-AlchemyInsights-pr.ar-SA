# <a name="solutions-for-unsupported-operating-system"></a>حلول "نظام التشغيل غير المدعوم"

عند محاولة تثبيت Microsoft 365، قد ترى رسالة خطأ **نظام التشغيل غير معتمد** إذا كان جهازك لا يفي بمتطلبات [النظام](https://products.office.com/office-system-requirements).
  
يتطلب Microsoft 365:
  
- الكمبيوتر: ويندوز 10، ويندوز 8.1، ويندوز 8، ويندوز 7 حزمة الخدمة 1، ويندوز سيرفر 2016، ويندوز سيرفر 2012 R2، ويندوز سيرفر 2012، أو ويندوز سيرفر 2008 R2

    او

- ماك OS X 10.10

إذا تلقيت هذه الرسالة أثناء محاولة تثبيت Office على جهاز محمول أو جهاز Chromebook، فراجع [كيفية تثبيت Microsoft Office وتشغيله على جهاز Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).

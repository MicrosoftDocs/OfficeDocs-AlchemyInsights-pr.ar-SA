# <a name="solutions-for-unsupported-operating-system"></a>حلول ل "نظام التشغيل غير معتمدة"

عند محاولة تثبيت Office 365/2016، قد تشاهد رسالة الخطأ التالية إذا كان الجهاز لا يلبي [متطلبات النظام](https://products.office.com/office-system-requirements).
  
 **"نظام التشغيل غير معتمدة"**
  
يتطلب office 365/2016:
  
الكمبيوتر: Windows 10 Windows 8.1، ويندوز 8، ويندوز 7 Service Pack 1، Windows Server 2016، Windows Server 2012 R2، Windows Server 2012 أو Windows Server 2008 R2
  
أو 
  
Mac OS X 10، 10
  
إذا حصلت على هذه الرسالة أثناء محاولة تثبيت Office على جهاز محمول أو تشروميبوك، راجع [إعداد البريد الإلكتروني على جهاز محمول وتطبيقات Office](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).

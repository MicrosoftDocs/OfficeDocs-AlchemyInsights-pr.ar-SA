# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>استخدام الشروط أو صيغ متقدمة في تدفق

يمكن تخصيص تدفق حيث ينفذ المهام فقط إذا كان شرط صحيحاً. لمزيد من المعلومات، راجع [إضافة شرط إلى تدفق](https://go.microsoft.com/fwlink/?linkid=872112).
  
لإنشاء صيغة متقدمة، راجع [مخطط لغة تعريف سير العمل لتطبيقات Azure المنطق](https://aka.ms/logicexpressions).

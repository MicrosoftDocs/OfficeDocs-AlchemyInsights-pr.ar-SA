# <a name="print-results-in-a-summary-of-responses"></a>طباعه النتائج في ملخص الاستجابات

يمكنك [طباعه نتائج Microsoft Forms](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc) كملخص للاستجابات أو لفرد واحد. 

لا يمكنك طباعه النماذج بدون استجابات ، ولكن يمكنك طلب أو التصويت علي هذه الميزة عن طريق زيارة [موقع UserVoice ل Microsoft forms](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box).

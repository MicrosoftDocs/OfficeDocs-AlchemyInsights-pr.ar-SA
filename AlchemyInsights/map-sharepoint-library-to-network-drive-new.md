# <a name="map-a-sharepoint-library-to-a-network-drive"></a>تعيين مكتبة SharePoint إلى محرك أقراص شبكة الاتصال

يتم تعيين مكتبة كمحرك أقراص شبكة المؤقتة والمعتمدة فقط من خلال Internet Explorer. أحياناً يجب فتح موقع SharePoint في برنامج Internet Explorer وتحديد **موقعة في البقاء** لمنع انتهاء مدة صلاحية جلسة العمل. بدلاً من ذلك، [مزامنة الملفات SharePoint باستخدام عميل المزامنة الجديدة أندريف](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> التي توفر [الملفات عند الطلب](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). الوصول إلى كافة الملفات في أندريف دون استخدام مساحة للتخزين المحلي.

إذا اخترت تعيين محرك أقراص بدلاً من [استخدام عميل المزامنة أندريف الجديد](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88)، تأكد من اتباع الخطوات الواردة في المقالة أدناه. 


**محركات أقراص الشبكة المعينة كيفية تكوين واستكشاف الأخطاء وإصلاحها**


راجع [تكوين ومحركات أقراص الشبكة المعينة لاستكشاف الأخطاء وإصلاحها](https://support.office.com/article/troubleshoot-mapped-network-drives-that-connect-to-sharepoint-online-ef399c67-4578-4c3a-adbe-0b489084eabe?ui=en-US&amp;rs=en-US&amp;ad=US).

ملاحظة: إذا كنت تستخدم Internet Explorer 10 مع Windows 7 أو Windows 8، وتلقى **رفض الوصول** أو **مسار لا يمكن الوصول إليه** عند تعيين محرك أقراص، قم بتثبيت [هذا الإصلاح العاجل](https://support.microsoft.com/help/2846960) لحل هذه المشكلة.

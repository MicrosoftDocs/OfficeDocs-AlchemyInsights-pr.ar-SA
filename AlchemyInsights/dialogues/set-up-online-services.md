# <a name="set-up-online-services"></a>إعداد الخدمات عبر الإنترنت

لإعداد الخدمات عبر الإنترنت، قم بتنفيذ الخطوات التالية:

1. على الصفحة **"إضافة السجلات" لموقعك** على ويب: أ. انقر **فوق "التالي"،** إذا لم يكن لديك موقع ويب. بعد ذلك، ست انتقل إلى صفحة **"تحديث إعدادات DNS".**
    ب. ابحث عن السجل لدى موفر استضافة DNS، إذا كان لديك موقع ويب. بعد ذلك، انسخه واللصق في الصفحة.
2. انقر **فوق** "التالي" للذهاب إلى صفحة **"تحديث إعدادات DNS".**

# <a name="set-up-and-configure-communication-compliance"></a>إعداد توافق الاتصالات وتكوينه

كمرجع مستقبلي، يجب أن يكون لدى المستخدمين الذين يتم مراقبتها بواسطة توافق الاتصالات ترخيص توافق Microsoft 365 E5 أو ترخيص Office 365 Enterprise E3.

* إذا كنت تريد إعداد توافق الاتصالات أو تكوينه، فشاهد [بدء العمل باستخدام توافق الاتصالات.](https://go.microsoft.com/fwlink/?linkid=2111549)
* إذا لم يكن لديك خطة Enterprise E5 حالية وتريد تجربة "الإشراف"، ف قم التسجيل للحصول على الإصدار التجريبي من [Office 365 Enterprise E5.](https://go.microsoft.com/fwlink/p/?LinkID=698279)

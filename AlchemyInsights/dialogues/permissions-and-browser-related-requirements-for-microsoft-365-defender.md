# <a name="permissions-and-browser-related-requirements-for-microsoft-365-defender"></a>الأذونات والمتطلبات المتعلقة بالمستعرض ل Microsoft 365 Defender

- الأذونات: تشغيل Microsoft 365 Defender، يجب أن تكون مسؤول عام أو مسؤول أمان في Azure Active Directory. لمشاهدة قائمة بالأدوار التي يمكنها استخدام Microsoft 365 Defender، ولمعرفة كيفية تنظيم الوصول إلى البيانات، يمكنك الرجوع إلى إدارة الوصول إلى [Microsoft 365 Defender.](https://go.microsoft.com/fwlink/?linkid=2143626)

- المتطلبات ذات الصلة بالمستعرض: للوصول إلى Microsoft 365 Defender في مركز أمان Microsoft 365، استخدم Microsoft Edge أو Internet Explorer 11 أو أي مستعرض ويب متوافق مع HTML 5.

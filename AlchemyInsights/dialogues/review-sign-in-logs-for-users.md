# <a name="review-sign-in-logs-for-users"></a>مراجعة سجلات تسجيل الدخول للمستخدمين

إليك كيفية القيام بذلك:

1. انتقل إلى **مراكز الإدارة**  >  **[في Azure Active Directory.](https://go.microsoft.com/fwlink/p/?linkid=2067268)**
1. حدد **تسجيل الدخول إلى Azure Active Directory.**  >  
1. إذا لم يكن لديك بالفعل اشتراك Azure AD P1 أو P2، فسوف يتم مطالبتك ببدء تشغيل الإصدار التجريبي المجاني المتميز. انقر فوق "بدء تجربة **Premium** مجانية" إذا أردت ذلك، ثم في المربع Azure AD Premium P2، **حدد**"تنشيط الإصدار التجريبي  >  **المجاني".** الفترة التجريبية المجانية هي 30 يوما.

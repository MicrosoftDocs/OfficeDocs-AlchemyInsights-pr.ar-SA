# <a name="set-up-safe-attachment-policies-in-microsoft-defender-for-office-365"></a>إعداد سياسات المرفقات الآمنة في Microsoft Defender ل Office 365

1. انتقل إلى مركز التوافق & الأمان في [Office 365](https://go.microsoft.com/fwlink/p/?linkid=2077143) **وحدد** المرفقات الآمنة  >    >  **لنص إدارة المخاطر.**
2. لتمكين هذه الميزة لأحمال العمل الأخرى، حدد **تشغيل Microsoft Defender ل Office 365** ل SharePoint Online و OneDrive و Microsoft Teams.
3. لإنشاء نهج جديد، حدد **الأيقونة "جديد** ( **+** ).
4. حدد اسم النهج ووصفه وإعداداته.
5. حدد **"حفظ".**

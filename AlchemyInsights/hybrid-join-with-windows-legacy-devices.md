# <a name="hybrid-join-with-windows-legacy-devices"></a>الانضمام المختلط مع الأجهزة القديمة لـ Windows

للحصول على إرشادات حول كيفية حل المشكلات المحتملة مع الأجهزة القديمة لـ Windows الدليل النشط لـ Azure المختلط، راجع [استكشاف أخطاء Azure Active Directory المختلط التي تم ربطها بالأجهزة ذات المستوى السفلي](https://docs.microsoft.com/azure/active-directory/devices/troubleshoot-hybrid-join-windows-legacy).

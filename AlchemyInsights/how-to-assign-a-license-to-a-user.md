<span data-ttu-id="fe601-p101">يحتاج مستخدم ترخيص بالوصول إلى الخدمات. فيما يلي الخطوات لإضافة أو تغيير أو إزالة ترخيص من مستخدم.</span><span class="sxs-lookup"><span data-stu-id="fe601-p101">A user needs a license to have access to services. 2. <span data-ttu-id="fe601-106">حدد المربع الموجود بجانب الاسم المستخدم الذي تريد تعيين أو تغيير أو إزالة ترخيص.</span><span class="sxs-lookup"><span data-stu-id="fe601-106">Select the box next to the name of the user to whom you want to assign, change, or remove a license.</span></span>

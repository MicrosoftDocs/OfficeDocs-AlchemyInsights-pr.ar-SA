1. <span data-ttu-id="96f73-104">[إعادة تعيين كلمة المرور الخاصة بالمستخدم](https://docs.microsoft.com/office365/admin/add-users/reset-passwords) فورًا.</span><span class="sxs-lookup"><span data-stu-id="96f73-104">[Reset the user's password](https://docs.microsoft.com/office365/admin/add-users/reset-passwords) immediately.</span></span> <span data-ttu-id="96f73-105">لا تقم بتوصيل كلمة المرور الجديدة عبر البريد الإلكتروني إلى المستخدم النهائي.</span><span class="sxs-lookup"><span data-stu-id="96f73-105">Do not communicate the new password through email to the end user.</span></span>

2. <span data-ttu-id="96f73-106">إزالة أي مريب [إعادة توجيه عناوين](https://docs.microsoft.com/office365/admin/email/configure-email-forwarding) تعيين على مستوى علبة البريد.</span><span class="sxs-lookup"><span data-stu-id="96f73-106">Remove any suspicious [forwarding addresses](https://docs.microsoft.com/office365/admin/email/configure-email-forwarding) set at the mailbox level.</span></span>

3. <span data-ttu-id="96f73-107">إزالة أي مريب [قواعد علبة الوارد](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) تعيين داخل علبة البريد.</span><span class="sxs-lookup"><span data-stu-id="96f73-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span>

4. <span data-ttu-id="96f73-108">إذا تم حظر المستخدم من إرسال البريد الإلكتروني، [انتقل إلى صفحة "المستخدمون المقيدون" لإلغاء حظر الحساب](https://protection.office.com/?hash=/restrictedusers).</span><span class="sxs-lookup"><span data-stu-id="96f73-108">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers).</span></span> <span data-ttu-id="96f73-109">بمجرد الانتهاء، سيكون بإمكان المستخدم استئناف إرسال الرسائل في غضون ساعة واحدة.</span><span class="sxs-lookup"><span data-stu-id="96f73-109">Once done, the user should be able to resume sending messages within 1 hour.</span></span>

5. <span data-ttu-id="96f73-110">إزالة حساب المستخدم من أي [مجموعات الأدوار الإدارية](https://docs.microsoft.com//office365/admin/add-users/assign-admin-roles) حتى تكون متأكداً من أن الحساب لم يعد مخترقاً.</span><span class="sxs-lookup"><span data-stu-id="96f73-110">Remove the user account from any [administrative role groups](https://docs.microsoft.com//office365/admin/add-users/assign-admin-roles) until you are confident that the account is no longer compromised.</span></span>

<span data-ttu-id="96f73-111">لتقليل احتمالية حدوث خرق للبيانات أو للحساب في المستقبل، نوصي بقراءة [ مخطط الأمان لـ Office 365](https://docs.microsoft.com//office365/securitycompliance/security-roadmap).</span><span class="sxs-lookup"><span data-stu-id="96f73-111">To minimize the potential of a data breach or a compromised account in the future, we recommend reading our [Office 365 Security best practices article](https://docs.microsoft.com//office365/securitycompliance/security-roadmap).</span></span>

1. <span data-ttu-id="7bf69-104">[إعادة تعيين كلمة المرور الخاصة بالمستخدم](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) فورًا.</span><span class="sxs-lookup"><span data-stu-id="7bf69-104">[Reset the user's password](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) immediately.</span></span> <span data-ttu-id="7bf69-105">لا تقم بتوصيل كلمة المرور الجديدة عبر البريد الإلكتروني إلى المستخدم النهائي.</span><span class="sxs-lookup"><span data-stu-id="7bf69-105">Do not communicate the new password through email to the end user.</span></span>

2. <span data-ttu-id="7bf69-106">إزالة أي مريب [إعادة توجيه عناوين](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) تعيين على مستوى علبة البريد.</span><span class="sxs-lookup"><span data-stu-id="7bf69-106">Remove any suspicious [forwarding addresses](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) set at the mailbox level.</span></span>

3. <span data-ttu-id="7bf69-107">إزالة أي مريب [قواعد علبة الوارد](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) تعيين داخل علبة البريد.</span><span class="sxs-lookup"><span data-stu-id="7bf69-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span>

4. <span data-ttu-id="7bf69-108">إذا تم حظر المستخدم من إرسال البريد الإلكتروني، [انتقل إلى "المستخدمون المقيدون" لإلغاء حظر الحساب](https://protection.office.com/?hash=/restrictedusers).</span><span class="sxs-lookup"><span data-stu-id="7bf69-108">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers).</span></span> <span data-ttu-id="7bf69-109">بمجرد الانتهاء، سيكون بإمكان المستخدم استئناف إرسال الرسائل في غضون ساعة واحدة.</span><span class="sxs-lookup"><span data-stu-id="7bf69-109">Once done, the user should be able to resume sending messages within 1 hour.</span></span>

5. <span data-ttu-id="7bf69-110">إزالة حساب المستخدم من أي [مجموعات الأدوار الإدارية](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) حتى تكون متأكداً من أن الحساب لم يعد مخترقاً.</span><span class="sxs-lookup"><span data-stu-id="7bf69-110">Remove the user account from any [administrative role groups](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) until you are confident that the account is no longer compromised.</span></span>

<span data-ttu-id="7bf69-111">لتقليل احتمالية حدوث خرق للبيانات أو للحساب في المستقبل، نوصي بالقراءة [ مقالة أفضل ممارسات الأمان لـ Office 365](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).</span><span class="sxs-lookup"><span data-stu-id="7bf69-111">To minimize the potential of a data breach or a compromised account in the future, we recommend reading our [Office 365 Security best practices article](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).</span></span>

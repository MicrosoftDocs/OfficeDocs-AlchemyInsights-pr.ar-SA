# <a name="edit-a-read-only-file"></a>تحرير ملف للقراءة فقط

في مكتبات المستندات، عند النقر فوق اسم المستند، يقوم برنامج Office قد بشكل غير متوقع فتحه كملف للقراءة فقط. لفتحه للتحرير، انقر فوق **فتح** (ثلاث نقاط)، ومن ثم انقر فوق **تحرير.**

# <a name="change-your-microsoft-account-picture"></a>تغيير صورة حساب Microsoft

1. انتقل إلى [معلومات حساب Microsoft .](https://go.microsoft.com/fwlink/p/?linkid=860841)
2. حدد **تغيير الصورة**.
3. حدد **صورة جديدة**.
4. اختر الصورة التي تريد استخدامها وحدد **فتح**.
5. اسحب مربع الصورة أو قم بتغيير حجمه وحدد **حفظ**.

يُرجى السماح بدخول التغييرات على صورة ملفك الشخصي حيز التنفيذ لمدة تصل إلى 24 ساعة.

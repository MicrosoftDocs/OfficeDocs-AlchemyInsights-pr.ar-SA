# <a name="change-your-account-picture"></a>تغيير صورة الحساب

1. انتقل إلى [معلومات الحساب الخاص بك ل Microsoft](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. ضمن الصورة، حدد **تغيير الصورة**.
3. حدد **الصورة الجديدة**.
4. اختيار الصورة التي تريد استخدامها، وحدد **فتح**.
5. سحب أو تغيير حجم المربع الصورة، وحدد **حفظ**.

الرجاء الانتظار 24 ساعة لحساب الصورة الفوتوغرافية تفعيل التغييرات.

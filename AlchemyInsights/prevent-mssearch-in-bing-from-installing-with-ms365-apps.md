# <a name="prevent-microsoft-search-in-bing-from-installing-with-microsoft-365-apps"></a>منع Microsoft Search في Bing من التثبيت باستخدام تطبيقات Microsoft 365

لمنع خدمة الخلفية ل Microsoft Search في Bing من التثبيت باستخدام تطبيقات Microsoft 365 للمؤسسات، استخدم أحد الخيارات التالية:

- أداة نشر Office أو نهج المجموعة
- Microsoft Endpoint Configuration Manager (الفرع الحالي) أو Microsoft Intune

لمعرفة المزيد، راجع كيفية منع تثبيت خدمة الخلفية ل Microsoft Search في Bing مع [تطبيقات Microsoft 365 للمؤسسات](https://go.microsoft.com/fwlink/?linkid=2151946).

# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>تعيين Microsoft Edge كمستعرض افتراضي على جهاز يعمل بنظام التشغيل Windows 10

إليك كيفية تنفيذ ذلك:

1. حدد زر **البدء،** ثم أدخل **التطبيقات الافتراضية**.

1. في نتائج البحث، حدد **التطبيقات الافتراضية**.

1. ضمن **مستعرض ويب،** حدد **Microsoft Edge**.

# <a name="change-your-profile-name"></a>تغيير اسم التشكيل الجانبي الخاص بك

1. انتقل إلى [معلومات الحساب الخاص بك ل Microsoft](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. حدد **تحرير اسم**. 
    
3. أدخل الاسم الأول واسم العائلة، وحدد **حفظ**. 
    
الرجاء الانتظار 24 ساعة لاسم التشكيل الجانبي الخاص بك حتى تسري التغييرات.
  
لمزيد من المعلومات، راجع [تغيير الصور الشخصية أو اسم في Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).

# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 تم رفض الوصول ، اكتشاف أساءه البريد العشوائي

تحدث هذه الرسالة عاده عند قيام المستخدمين بإرسال رسائل البريد الكتروني من عناوين IP باستخدام مجال *onmicrosoft.com* الذي تم تعيينه لمستاجرين جدد في Microsoft 365. تتمثل أسهل طريقه لحل هذه المشكلة في:

1. [أضف مجالا إلى المستاجر الخاص بك](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain).

2. قم [بتغيير عنوان البريد الكتروني الأساسي للمستخدمين](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address) إلى المجال المخصص الجديد الذي أضفته للتو.

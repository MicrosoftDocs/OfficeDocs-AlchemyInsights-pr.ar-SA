# <a name="manage-recurring-billing"></a>إدارة الفواتير المتكررة

يمكنك تكرار الفوترة أو إيقاف تشغيله في مركز مسؤول Microsoft 365.
  
1. في مركز الإدارة، انتقل إلى **الدفع** \> صفحة [& المنتجات أو الخدمات](https://go.microsoft.com/fwlink/p/?linkid=842054) .

2. ضمن **الفوترة**، المجاورة **تجدد على** أو **اكسبيرينج على**، حدد **تحرير**.

3. في لوحة **إعدادات الفوترة المتكررة** ، حدد **إيقاف التشغيل** لإيقاف تشغيله. إذا تم مسبقاً إيقاف تشغيل المتكررة الفواتير، حدد **شهريا** أو **مرة واحدة** لتشغيله.

4. حدد **تغيير الإرسال**.

لمعرفة المزيد، راجع [تشغيل تكرار الفوترة أو إيقاف تشغيله](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/renew-your-subscription?view=o365-worldwide#turn-recurring-billing-off-or-on).

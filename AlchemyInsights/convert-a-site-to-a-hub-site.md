# <a name="create-a-sharepoint-hub-site"></a>إنشاء موقع SharePoint لوحة الوصل

1. في مركز مسؤول SharePoint جديدة ضمن **مواقع**، انقر فوق **مواقع نشطة**.

2. حدد الموقع الذي تريده لموقع لوحة وصل، انقر فوق **موقع لوحة الوصل**، ثم انقر فوق **السجل كموقع لوحة الوصل**.

3. أدخل اسم عرض لموقع لوحة الوصل وتحديد المستخدمين الفرديين أو مجموعات الأمان التي تريد السماح لربط مواقع مع لوحة الوصل.

    > [!NOTE]
    >  نوصي بتحديد موقع اتصال أو موقع فريق يستخدم القالب الجديد. إذا كنت تستخدم موقع فريق التقليدية، تنقل لوحة الوصل ستظهر فقط في الصفحات الحديثة، وإعدادات الموقع لوحة الوصل ستظهر فقط على الصفحات الحديثة. لا يمكن تسجيل مواقع بالفعل المقترنة مع محور آخر كموقع لوحة وصل.

# <a name="merge-microsoft-accounts"></a>دمج حسابات Microsoft

- لا يمكن دمج حساب Microsoft وحسابات العمل أو المؤسسة التعليمية لـ Office 365 وإنشاء ارتباط بينهما في خدمات مختلفة، ولكن يمكن استخدامها معاً.
- لا يمكن دمج حساب Microsoft أو ربطه (حتى مع عناوين بريد إلكتروني منفصلة) أو حساب غير تابع لـ Microsoft (مثل حساب Gmail) بحساب Microsoft موجود.

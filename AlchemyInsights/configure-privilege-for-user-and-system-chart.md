# <a name="configure-privilege-for-user-and-system-chart"></a>تكوين امتياز لمخطط المستخدم والنظام

**العرض**: لا يوجد امتياز قراءه للمخطط.

**السبب**: لا توجد امتيازات كافيه أو لا تحتفظ بأخطاء الامتيازات الضرورية.

**الحل**: تكوين امتياز للمخطط

**الخطوات**:

1. انتقل إلى **الإعدادات-> التخصيص-> تخصيص ادوار الأمان الخاصة ب> النظام**.

2. انقر نقرا مزدوجا فوق الدور.

    - يكون تكوين امتياز مخطط المستخدم ضمن علامة التبويب **السجلات الاساسيه** .
    - يكون تكوين امتياز مخطط النظام ضمن علامة التبويب " **تخصيص** ".

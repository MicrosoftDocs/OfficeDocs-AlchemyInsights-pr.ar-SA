# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>تحويل علبة بريد مستخدم إلى علبة بريد الغرفة أو المعدات

يمكنك أيضا تحويل علبة بريد من نوع علبة بريد واحدة إلى أخرى، ولكن يجب عليك القيام بهذه المهام باستخدام PowerShell Exchange عبر إنترنت. راجع [تحويل علبة بريد](https://go.microsoft.com/fwlink/p/?LinkId=832875) للحصول على مزيد من التفاصيل.

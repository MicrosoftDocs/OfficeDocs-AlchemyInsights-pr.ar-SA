# <a name="restore-a-deleted-user"></a>استعاده مستخدم محذوف

استعاده مستخدم باستخدام الخطوات التالية:
  
1. انتقل إلى [ \> المستخدمين المحذوفين](https://admin.microsoft.com/adminportal/home#/deletedusers).

2. اختر المستخدم ، ثم حدد **استعاده**.

3. اتبع المطالبات لتعيين كلمه المرور.

4. انقر فوق **إرسال بريد الكتروني وإغلاق**، وقد انتهت العملية!

الم تكن سهله ؟ للحصول علي مزيد من التفاصيل والخطوات باستخدام لقطات الشاشة ، راجع هذه المقالة: [استعاده مستخدم](https://docs.microsoft.com/microsoft-365/admin/add-users/restore-user). إذا كنت تعلم انك تحتاج إلى استعاده علبه بريد المستخدم ، فراجع: [حذف علب بريد المستخدمين أو استعادتها في Exchange Online](https://docs.microsoft.com/exchange/recipients-in-exchange-online/delete-or-restore-mailboxes).

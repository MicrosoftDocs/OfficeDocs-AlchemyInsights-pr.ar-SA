# <a name="create-a-new-folder"></a>إنشاء مجلد جديد

1. ضمن **المجلدات** في "جزء التنقل"، حدد **مجلد جديد**.

2. اكتب اسماً للمجلد، ثم اضغط Enter.

لمزيد من المعلومات، راجع [إنشاء مجلد في Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).

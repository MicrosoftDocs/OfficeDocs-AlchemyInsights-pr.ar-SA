# <a name="learn-about-mapping-between-google-chromes-and-microsoft-edges-policies"></a>تعرف على المزيد حول التعيين بين سياسات Google Chrome و Microsoft Edge

يتم تعيين سياسات Google Chrome إلى سياسات Microsoft Edge ذات الصلة المعتمدة في الإصدار 80 من المستعرض. لمزيد من المعلومات، راجع [تعيين نهج Google Chrome إلى Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-chrome-to-newedge).

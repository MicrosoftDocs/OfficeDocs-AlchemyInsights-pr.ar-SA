# <a name="submit-microsoft-cloud-for-healthcare-support-requests"></a>إرسال سحابه Microsoft لطلبات دعم الرعاية الصحية

سحابه Microsoft لعملاء الرعاية الصحية يرجى إرسال بطاقتك عبر [مركز أداره النظام الأساسي للطاقة](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow) الجديد.

يوفر المدخل الجديد مسارا مباشرا أكثر إلى الدعم المساعد والذي يتم تحسينه لسحابه Microsoft للحصول علي الرعاية الصحية.

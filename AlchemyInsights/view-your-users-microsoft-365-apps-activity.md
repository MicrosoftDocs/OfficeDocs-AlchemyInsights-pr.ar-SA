# <a name="view-your-users-microsoft-365-apps-activity"></a>عرض نشاط تطبيقات Microsoft 365 للمستخدمين

يوفر [لك تقرير تنشيط Office](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide) عرضًا للمستخدمين الذين قام بتثبيت تطبيقات Microsoft 365 وتنشيطها على جهاز واحد على الأقل.

بالإضافة إلى ذلك، يمكنك الحصول على عرض في نشاط تطبيقات Microsoft 365 للمستخدمين باستخدام [تقرير استخدام تطبيقات Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide).

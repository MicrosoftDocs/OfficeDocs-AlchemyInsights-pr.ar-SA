# <a name="how-to-type-special-characters-on-a-mac"></a>كيفية كتابة أحرف خاصة في نظام التشغيل Mac

لكتابة أحرف خاصة على ماك، فتح قائمة **الأحرف الخاصة** باختصار لوحة المفاتيح ⌘ **الأمر** + **عنصر التحكم** + **الفضاء**.

يعتبر سريع اختصار @ **ALT** + **64**.

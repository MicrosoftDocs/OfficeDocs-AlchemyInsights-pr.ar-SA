# <a name="use-whiteboard-with-microsoft-365"></a>استخدام لوح المعلومات مع Microsoft 365

إن Microsoft Whiteboard هو لوحة رقمية حرة يجتمع من خلالها الأشخاص والمحتوى والأفكار معاً. 

- [تمكين لوح المعلومات](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_07) كمسؤول Microsoft 365. 

- استخدام [لوح المعلومات مع Microsoft Teams](https://support.microsoft.com/ar-SA/office/use-whiteboard-in-microsoft-teams-7a6e7218-e9dc-4ccc-89aa-b1a0bb9c31ee). 

- [التعاون](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920#bkmk_27) مع شخص آخر في لوح المعلومات. 

للحصول على مزيد من المعلومات، اطلع على [تعليمات Microsoft Whiteboard](https://support.office.com/article/Microsoft-Whiteboard-Help-d236aef8-fcdf-4b5e-b5d7-7f157461e920).

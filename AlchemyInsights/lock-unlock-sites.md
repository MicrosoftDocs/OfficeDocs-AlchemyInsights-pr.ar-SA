# <a name="lock-and-unlock-sites"></a>تأمين المواقع وإلغاء تأمينها

كمسؤول Microsoft 365 أو مسؤول SharePoint، يمكنك حظر الوصول إلى موقع أو جعل الموقع للقراءة فقط باستخدام Microsoft PowerShell لتغيير حالة تأمين الموقع. لمزيد من المعلومات، راجع [تأمين المواقع وإلغاء تأمينها](/sharepoint/manage-lock-status).

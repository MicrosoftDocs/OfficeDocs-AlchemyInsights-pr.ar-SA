# <a name="change-your-microsoft-365-billing-currency"></a>تغيير Microsoft 365 الفوترة

لا يمكنك تغيير عملة الفوترة لاشتراكك. ترتبط العملة للبلد أو المنطقة التي تم تحديدها عند Microsoft 365. لتغيير بلد مؤسستك أو إقليمك، سجل للحصول على حساب جديد، واختر البلد أو المنطقة المطلوبة، ثم قم بشراء اشتراك جديد. 

لتغيير تفاصيل عنوان مؤسستك، راجع تغيير عنوان المؤسسة والاتصال [التقني والمزيد.](/microsoft-365/admin/manage/change-address-contact-and-more) لتغيير عنوان الاشتراك أو الفوترة المقترن، راجع تغيير عناوين الفوترة [Microsoft 365 للأعمال](/microsoft-365/commerce/billing-and-payments/change-your-billing-addresses).

# <a name="deploy-microsoft-365"></a>نشر Microsoft 365

يوفر " [مرشد النشر ل Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2072646) " للعملاء التجاريين الإرشادات عند اعداد أدوات الانتاجيه ونهج الأمان وإمكانيات أداره الاجهزه. باستخدام اشتراك Microsoft 365 Business Premium أو Microsoft 365 للمؤسسة ، يمكنك استخدام هذا المرشد لاعداد أجهزه المؤسسة وتكوينها.

ستتلقى الإرشادات والوصول إلى الموارد لتمكين خدمات السحابة وتحديث الاجهزه إلى أحدث إصدار معتمد من Windows 10 والانضمام إلى الاجهزه إلى Azure Active directory (Azure AD) ، كل ذلك في موقع مركزي واحد.

# <a name="domain-transfers"></a>عمليات نقل المجال

[كيفية نقل مجال Microsoft الذي تم شراؤه إلى موفر آخر بعد مرور 60 يوماً](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/transfer-a-domain-from-microsoft-to-another-host).

- يرجى ملاحظه انه لا يمكن نقل مجال Microsoft الذي تم شراؤه بين Microsoft 365 مستاجر.
- علي الرغم من ان المجالات التي اشتريتها Microsoft لا تعتمد تغيير سجلات NS ، فضع في اعتبارك [تحديث سجلات DNS لموقعك علي](https://docs.microsoft.com/microsoft-365/admin/dns/update-dns-records-to-retain-current-hosting-provider?view=o365-worldwide) ويب بدلا من تحويل تسجيل المجالات إلى مضيف الويب.

يمكن [التحقق من المجالات المخصصة واستخدامها مع Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain?view=o365-worldwide).

- لا يمكن نقل تسجيل المجالات المخصصة لجهة خارجيه أو الفوترة إلى Microsoft.

لا يمكن نقل المجالات الافتراضية Onmicrosoft.com أو أعاده تسميتها.

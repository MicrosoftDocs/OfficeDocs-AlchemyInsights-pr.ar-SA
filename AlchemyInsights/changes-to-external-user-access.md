# <a name="changes-to-external-user-access"></a>التغييرات لوصول المستخدم الخارجي

ابتداء من 23 آذار/مارس 2018، المستخدمين الخارجيين لن تكون قادراً على الوصول إلى المحتوى الذي تمت مشاركته مع الجميع كافة المستخدمين المصادق عليهم أو كافة نماذج المستخدمين، حتى إذا دعت أو إضافة إلى مجموعات المأذون به. سوف يكون المستخدمون الخارجيون فقط قادراً على الوصول إلى المحتوى الذي تمت مشاركته مع المستخدم أو مع المجموعات التي ينتمي إليها المستخدم. إذا كنت تريد الاحتفاظ بالإعدادات الموجودة، يرجى الاطلاع على:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365

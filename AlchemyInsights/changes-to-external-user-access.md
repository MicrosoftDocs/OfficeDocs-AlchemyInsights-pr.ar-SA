# <a name="changes-to-external-user-access"></a>التغييرات في وصول المستخدمين الخارجيين

بدءا من 23 مارس 2018 ، لن يتمكن المستخدمون الخارجيون من الوصول إلى المحتوي الذي تمت مشاركته مع الجميع أو كل المستخدمين المصادق عليهم أو مستخدمي النماذج ، حتى لو تمت دعوتهم أو اضافتهم إلى المجموعات المخولة. سيتمكن المستخدمون الخارجيون فقط من الوصول إلى المحتوي الذي تمت مشاركته مع المستخدم أو المجموعات التي ينتمي اليها المستخدم. إذا كنت ترغب في الاحتفاظ بالإعدادات الموجودة ، فالرجاء مراجعه: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365

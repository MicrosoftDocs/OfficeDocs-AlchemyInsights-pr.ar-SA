# <a name="changes-to-external-user-access"></a>التغييرات على وصول المستخدم الخارجي

بدءًا من 23 مارس 2018، لن يتمكن المستخدمون الخارجيون بعد الآن من الوصول إلى المحتوى الذي تمت مشاركته مع الجميع أو جميع المستخدمين الموثقين أو كافة مستخدمي النماذج، حتى إذا تمت دعوتهم أو إضافتهم إلى المجموعات المعتمدة. سيتمكن المستخدمون الخارجيون فقط من الوصول إلى المحتوى الذي تمت مشاركته مع المستخدم أو مع المجموعات التي ينتمي إليها المستخدم. إذا كنت ترغب في الاحتفاظ بالإعدادات الحالية، يرجى الاطلاع على:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365

# <a name="calendar-publishing-information"></a>معلومات نشر التقويم

- يمكن لمسؤولي Microsoft 365 تمكين نشر التقويم ومشاركة التقويم للمؤسسة أو تعطيلهما. لمزيد من المعلومات، راجع [تشغيل نشر التقويم حتى يتمكن الأشخاص من مشاركة المعلومات](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC) ومشاركة [التقويمات مع المستخدمين الخارجيين](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).

- يمكن لمستخدمي Outlook مشاركة التقويمات الخاصة بهم. لمزيد من المعلومات، راجع [مشاركة تقويم Outlook مع أشخاص آخرين](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88).

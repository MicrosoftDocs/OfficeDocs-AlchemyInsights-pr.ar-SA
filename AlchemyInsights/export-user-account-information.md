# <a name="export-user-account-information"></a>تصدير معلومات حساب المستخدم

لتصدير معلومات حساب المستخدم في مركز إدارة Microsoft 365، بما في ذلك معلومات حول التراخيص وسياسة كلمة المرور والحسابات المحظورة وحالة مزامنة AAD والمزيد، انتقل إلى صفحة **المستخدمين** > [النشطين،](https://go.microsoft.com/fwlink/p/?linkid=834822) ثم حدد **مستخدمي التصدير**.

# <a name="add-permissions"></a>أضافه أذونات

1. استخدم نقطه نهاية موافقه المسؤول [لتكوين قائمه الأذونات المطلوبة بشكل ثابت لأحد التطبيقات](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application).
1. هناك اثنين من الأذونات المتوفرة لمنح القدرة علي إنشاء تسجيلات التطبيقات ، كل منها ذات سلوك مختلف. لمزيد من المعلومات ، راجع [أذونات تسجيل التطبيق للأدوار المخصصة في Azure Active](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions)directory.
1. [أذونات الدور](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions): الجداول التي تصف الأذونات المحددة في Azure Active directory المحدد لكل دور. قد يكون لدي بعض الأدوار أذونات اضافيه في Microsoft services خارج Azure Active directory.

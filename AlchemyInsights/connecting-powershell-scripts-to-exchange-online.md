# <a name="connecting-powershell-scripts-to-exchange-online"></a>توصيل البرامج النصية PowerShell إلى Exchange عبر الإنترنت

المصادقة الأساسية في Exchange عبر الإنترنت سوف يتم إهمال و الطريقة إلى الأمام الاتصال باستخدام المصادقة المستندة إلى شهادة البرامج النصية والمهام غير المراقب. لمعرفة المزيد، راجع [مصادقة التطبيقات فقط للبرامج النصية غير المراقب في الوحدة النمطية EXO V2](https://docs.microsoft.com/powershell/exchange/app-only-auth-powershell-v2).

# <a name="desktop-app-for-sharepoint"></a>تطبيق سطح المكتب ل SharePoint

يتم إدخال مؤسستك في برنامج المعاينة الخاص لتطبيق سطح المكتب ل SharePoint. لا تعتمد Microsoft حاليا تطبيق سطح المكتب ل SharePoint (المعاينة الخاصة). يرجى استخدام شركاء المعاينة الخاصة في Microsoft المدرجين للحصول علي مساعده باستخدام تطبيق سطح المكتب ل SharePoint.

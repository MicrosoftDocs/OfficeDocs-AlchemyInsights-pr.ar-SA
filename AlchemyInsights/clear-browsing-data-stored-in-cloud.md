# <a name="clear-the-browsing-data-stored-in-the-cloud-if-youve-turned-on-sync-or-personalization"></a>مسح بيانات الاستعراض المخزنة في السحابة (إذا قمت ب تشغيل المزامنة أو تخصيصها)

لعرض البيانات التي حفظتها في السحابة أو إدارتها أو حذفها، استخدم لوحة معلومات الخصوصية. لن يتم حذف البيانات التي تحذفها باستخدام لوحة المعلومات من جهازك. للحصول على التفاصيل، راجع [عرض البيانات على لوحة معلومات الخصوصية](https://support.microsoft.com/windows/view-your-data-on-the-privacy-dashboard-03d3e27f-1981-5ff4-ba1c-d6b1031ae433).

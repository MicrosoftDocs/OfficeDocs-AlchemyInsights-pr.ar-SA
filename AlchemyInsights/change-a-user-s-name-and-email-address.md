# <a name="change-a-users-name-and-email-address"></a>تغيير اسم المستخدم وعنوان البريد الإلكتروني

من السهل تغيير اسم المستخدم وعنوان البريد الإلكتروني في Microsoft 365. انتقل إلى **Users** صفحة \> [المستخدمين النشطين](https://go.microsoft.com/fwlink/p/?linkid=834822) وحدد المستخدم لـ تحرير معلوماتهم. يمكنك أيضًا استخدام هذه المقالة للحصول على إرشادات خطوة بخطوة:
  
[تغيير اسم مستخدم وعنوان بريد إلكتروني في Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address)
  
 **ملاحظة:** إذا قمت بتغيير عنوان البريد الإلكتروني الرئيسي للمستخدم، سيظل عنوان بريده الإلكتروني القديم صالحًا لتلقي البريد الإلكتروني (يحدث هذا تلقائيًا). أيضاً، إذا كانت مؤسستك في حالة الهجين، يجب إجراء كافة تغييرات البريد الإلكتروني واسم المستخدم في الدليل النشط الداخلي ثم مزامنتها مع السحابة.

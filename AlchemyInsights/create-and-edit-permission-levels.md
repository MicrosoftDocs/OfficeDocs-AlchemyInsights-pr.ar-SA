# <a name="working-with-sharepoint-permission-levels"></a>العمل مع مستويات أذونات SharePoint

يأتي SharePoint بمستويات أذونات افتراضية يمكنك استخدامها، مثل **المساهمة** **وعرض فقط**. ولكن إذا كان أحد هذه لا يلبي احتياجاتك، يمكنك إنشاء مستوى إذن جديد. ضع في اعتبارك أنه من الأسهل تتبع مستويات الأذونات إذا لم تقم بتغيير الإعدادات الافتراضية.

[كيفية إنشاء مستويات الأذونات وتحريرها](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)

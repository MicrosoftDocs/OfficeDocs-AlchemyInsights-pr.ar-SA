# <a name="modern-commerce-administrator-role"></a>دور مسؤول التجارة الحديثة

يتم تعيين هذا الدور تلقائيًا للمستخدمين الذين يشترون لمنحهم وصولًا محدودًا إلى بوابة المسؤول لإدارة عملية الشراء هذه. هؤلاء المستخدمين ليسوا مسؤولين وليس لديهم امتيازات مرتفعة.

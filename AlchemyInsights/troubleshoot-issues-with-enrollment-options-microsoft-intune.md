# <a name="troubleshoot-issues-with-enrollment-options-microsoft-intune"></a>استكشاف المشاكل المتعلقة بخيارات التسجيل Microsoft Intune

راجع الموارد المذكورة أدناه لحل مشكلتك الآن. إذا لم تساعد هذه الموارد، فاتصل بالدعم بالنقر [هنا](https://portal.azure.com/#blade/Microsoft_Intune_DeviceSettings/ExtensionLandingBlade/help) واختيار "إنشاء طلب دعم". 
  
- [مراجعة Intune TechNet للعثور على إجابات وحلول للقضايا الشائعة.](https://social.technet.microsoft.com/Forums/home?category=microsoftintune&amp;filter=alltypes&amp;sort=lastpostdesc)
    
- استخدم [بوابة استكشاف الأخطاء وإصلاحها Intune](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) لتشخيص حالات فشل التسجيل الشائعة وحلها. راجع [هذا المستند](https://docs.microsoft.com/intune/help-desk-operators) لمزيد من التفاصيل. 
    
- راجع [هذا المستند](https://docs.microsoft.com/intune-classic/Troubleshoot/troubleshoot-device-enrollment-in-intune) للحصول على قائمة بالأخطاء الشائعة التي تمنع التسجيل والحلول لكل منها. 
    
- [تعرف على خيارات التسجيل في Microsoft Intune](https://docs.microsoft.com/intune/enrollment-options).

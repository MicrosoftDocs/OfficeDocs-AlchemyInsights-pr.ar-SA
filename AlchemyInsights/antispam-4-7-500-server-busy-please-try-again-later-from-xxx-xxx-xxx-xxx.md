# <a name="47500-server-busy-please-try-again-later"></a>4.7.500 خادم مشغول، يرجى المحاولة مرة أخرى في وقت لاحق

يحدث هذا الخطأ عندما يتجاوز حجم حركة مرور البريد الإلكتروني من عنوان IP المصدر الحد استنادًا إلى سمعة (أو عدم وجود سمعة) لعنوان IP المصدر.

سيتم إنهاء صلاحية حظر البريد الإلكتروني من عنوان IP المصدر في غضون ساعة. إذا كان عنوان IP المصدر هو خادم بريد إلكتروني داخلي ينتمي إليك، فتحقق من تكوين موصل تدفق البريد. إذا استمر السلوك لأكثر من ساعة، اتصل بالدعم لطلب استثناء لعنوان IP المصدر.

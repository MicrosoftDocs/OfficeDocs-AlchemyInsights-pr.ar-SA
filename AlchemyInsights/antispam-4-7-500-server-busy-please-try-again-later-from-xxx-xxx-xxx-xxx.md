# <a name="47500-server-busy-please-try-again-later"></a>4.7.500 Server مشغول ، يرجى المحاولة مره أخرى لاحقا

يحدث هذا الخطا عندما يتجاوز حجم نقل بيانات البريد الكتروني من عنوان IP المصدر الحد الذي يعتمد علي سمعه عنوان IP المصدر أو نقصه فيه.

ستنتهي صلاحيه حظر البريد الكتروني من عنوان IP المصدر في غضون ساعة. إذا كان عنوان IP المصدر هو خادم البريد الكتروني المحلي الذي ينتمي اليك ، فتحقق من تكوين موصل تدفق البريد. إذا استمر السلوك لأكثر من ساعة ، فاتصل بقسم الدعم لطلب استثناء لعنوان IP المصدر.

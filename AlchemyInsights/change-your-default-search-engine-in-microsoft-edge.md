# <a name="change-your-default-search-engine-in-microsoft-edge"></a>تغيير محرك البحث الافتراضي في Microsoft Edge

لتغيير محرك البحث الافتراضي، اتبع الإرشادات التالية:
1. في شريط العنوان، قم بإجراء بحث باستخدام محرك البحث الذي تريد تعيينه كافتراضي.
2. حدد **... (الإعدادات والمزيد) > الإعدادات > الخصوصية والبحث والخدمات**.
3. قم بالتمرير وصولا إلى مقطع **الخدمات،** ثم حدد **شريط العنوان وابحث .**
4. من محرك **البحث المستخدم في قائمة شريط العنوان،** حدد محرك البحث المفضل لديك.

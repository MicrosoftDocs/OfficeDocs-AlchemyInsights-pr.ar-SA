# <a name="enable-multi-factor-authentication"></a>تمكين المصادقة متعددة العوامل

لتمكين المصادقة متعددة العوامل في مركز إدارة Microsoft 365:

1. انتقل إلى صفحة **المستخدمين** \> **النشطين.**
    
2. في أعلى الصفحة، حدد **المصادقة متعددة العوامل**. 
    
3. ابحث عن المستخدم أو المستخدمين الذين تريد تمكينهم للمصادقة متعددة العوامل. لمشاهدة كافة المستخدمين، قد تحتاج إلى تغيير عامل **تصفية حالة Auth متعدد العوامل** في الأعلى.
    
4. حدد خانة الاختيار بجوار المستخدمين الذين تريد تمكينهم.
    
5.  في جزء معلومات المستخدم الصحيح، ضمن **الخطوات السريعة،** حدد **تمكين**. 
    
6. في مربع الحوار الذي يفتح، حدد **تمكين auth متعدد العوامل**.

# <a name="enable-versioning-for-a-list-or-library"></a>تمكين تعيين إصدار لقائمة أو مكتبة

1. انتقل إلى القائمة أو المكتبة.
    
2. انقر فوق **إعدادات**، وانقر فوق **إعدادات القائمة**أو **إعدادات مكتبة** .
    
3. انقر فوق **إعدادات تعيين الإصدار**.
    
4. حدد الإعدادات التي تريدها، ومن ثم انقر فوق **موافق**.

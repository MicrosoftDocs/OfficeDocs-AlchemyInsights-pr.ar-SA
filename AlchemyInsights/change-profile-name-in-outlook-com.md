# <a name="change-your-profile-name"></a>تغيير اسم ملفك الشخصي

1. انتقل إلى [معلومات حساب Microsoft .](https://go.microsoft.com/fwlink/p/?linkid=860841)
    
2. حدد **اسم تحرير**. 
    
3. أدخل الاسم الأول واسم العائلة، وحدد **حفظ**. 
    
يُرجى السماح بسريان التغييرات على اسم ملفك الشخصي لمدة تصل إلى 24 ساعة.
  
لمعرفة المزيد، راجع [تغيير صورة ملفك الشخصي أو اسمك في Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).

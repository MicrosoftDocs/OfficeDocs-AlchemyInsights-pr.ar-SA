# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>إصلاح أخطاء "فشل المصادقة" في تدفق Microsoft

إذا تلقيت رسالة خطأ "فشل المصادقة" عند محاولة تسجيل الدخول إلى Microsoft Flow، يكون بسبب تعطيل خدمة تدفق في الدليل النشط Azure. للحصول على تفاصيل وخطوات لإصلاح الخطأ، راجع [لا يمكنك تسجيل الدخول إلى Microsoft Flow وتتلقى خطأ "فشل المصادقة".](https://support.microsoft.com/help/4316891)

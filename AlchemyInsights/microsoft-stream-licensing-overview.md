# <a name="microsoft-stream-licensing-overview"></a>نظرة عامة حول ترخيص Microsoft Stream

Microsoft Stream هي خدمة سحابية مرخصة على أساس اشتراك كل مستخدم. وهي متوفرة كجزء من اشتراكات Microsoft 365 المؤهلة. راجع أسعار Microsoft 365 للحصول على معلومات حول كل نوع اشتراك.

- [نظرة عامة حول ترخيص Microsoft Stream](https://docs.microsoft.com/stream/license-overview)

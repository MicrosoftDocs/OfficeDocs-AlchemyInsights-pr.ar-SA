# <a name="webpage-html-web-resources-limitations"></a>قيود الموارد ويب "صفحة ويب" (HTML)

* مورد ويب HTML لا يحتوي على أية تعليمات برمجية التي يجب تنفيذها على الخادم. يتعذر تحميل صفحات ASP.NET كموارد ويب HTML.

* موارد ويب HTML فقط قبول عدد محدود من معاملات سلسلة الاستعلام. مزيد من المعلومات: [تمرير المعلمات إلى موارد ويب HTML](https://docs.microsoft.com/en-us/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)

# <a name="use-microsoft-edge-to-clear-the-browsing-data-stored-by-internet-explorer"></a>استخدام Microsoft Edge لمسح بيانات الاستعراض المخزنة بواسطة Internet Explorer

تتوفر هذه الميزة فقط إذا كانت مؤسستك قد قمت تشغيل وضع Internet Explorer (IE). تجدر الإشارة إلى أن مسح بيانات استعراض IE لن يؤثر على بيانات الاستعراض في Microsoft Edge.
*
1. في Microsoft Edge، حدد **الإعدادات والمزيد ...**  >  **الإعدادات**  >  **الخصوصية والخدمات.**

1. ضمن **مسح بيانات الاستعراض ل Internet Explorer،** حدد اختيار ما يجب **مسحه.**

1. حدد أنواع البيانات التي تريد مسحها.

1. حدد **مسح الآن**.

# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>رفض الوصول 550 5.7.705، تجاوز عتبة المستأجر

تحدث هذه الرسالة عندما تم إرسال البريد العشوائي أو المجمع كثيرا بمؤسستك ونضعها كتلة البريد الصادر.
الأسباب شيوعاً هي حسابات المسؤول المكسور التي تم استخدامها لإنشاء روابط أو خوادم الداخلية المكسور. أما شرط السماح للبريد الإلكتروني العشوائي بالمرور عبر المؤسسة.

لإزالة هذه الكتلة، تحتاج إلى فهم وشرح السبب عامل دعم، بالإضافة إلى حل المشكلة الأساسية.
انقر فوق **الاتصال بنا** أدناه عندما تكون مستعدا.

# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>رفض الوصول 550 5.7.705، تجاوز عتبة المستأجر

تحدث هذه الرسالة عندما تم إرسال البريد العشوائي أو المجمع كثيرا بمؤسستك Office 365. لإزالة هذه الكتلة، تحتاج إلى فهم وشرح السبب عامل دعم، بالإضافة إلى حل المشكلة الأساسية.

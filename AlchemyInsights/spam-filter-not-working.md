# <a name="spam-filter-not-working"></a>لا يعمل عامل تصفية البريد الإلكتروني العشوائي

إذا كنت تعتقد رسالة تلقيتها في Outlook.com الخداع أو التحايل خدعة، حدد الرسالة وحدد **العشوائي** \> **الخداع** لوضع علامة عليها كخدعة تصيد. لمزيد من المعلومات حول الاعتداء والخداع والغش، [اقرأ هذا](https://support.office.com/article/0d882ea5-eedc-4bed-aebc-079ffa1105a3?wt.mc_id=Office_Outlook_com_Alchemy). لحظر عناوين البريد الإلكتروني أو إضافة المجالات أو العناوين إلى قائمة المرسلين الموثوق بهم، أو تطبيق خيارات البريد الإلكتروني غير الهامة الأخرى، حدد **خيارات البريد الإلكتروني العشوائي**.

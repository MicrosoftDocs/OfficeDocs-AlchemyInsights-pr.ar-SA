# <a name="access-services"></a>Access Services

يتم سحب ميزة خدمات الوصول في SharePoint Online. لم يعد من الممكن إنشاء تطبيقات ويب جديدة قائمة على Access وقواعد بيانات الويب ، وسيتم إيقاف تشغيل تطبيقات الويب وقواعد بيانات الويب الحالية بحلول أبريل 2018. لا تتأثر قواعد بيانات سطح المكتب Access (.accdb) بهذا التغيير. نوصي Microsoft PowerApps كبديل لتطبيقات الويب Access وقواعد بيانات الويب. 
  
[تعرف على Microsoft PowerApps](https://powerapps.microsoft.com/)

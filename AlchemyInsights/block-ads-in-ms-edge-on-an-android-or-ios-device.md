# <a name="block-ads-in-microsoft-edge-on-an-adroid-or-ios-device"></a>حظر الإعلانات في Microsoft Edge علي جهاز Android أو iOS

لحظر كل الإعلانات باستثناء تلك التي تعتبر مقبوله بواسطة AdBlock بالاضافه إلى:
1. تحديد **...** > **إعدادات**  >  **أدوات حظر المحتوي**.
2. شغل كتل الإعلانات.

لحظر كل الإعلانات:
1. تحديد **...** > **إعدادات**  >  **أدوات حظر المحتوي**.
2. شغل كتل الإعلانات.
3. حدد **إعدادات متقدمة**.
4. إيقاف تشغيل السماح بإعلانات مقبوله.

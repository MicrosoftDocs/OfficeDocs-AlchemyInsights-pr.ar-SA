# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: استكشاف الأخطاء وإصلاحها ' معالجه التغييرات '

إذا رايت "معالجه التغييرات" لفتره ممتدة من الوقت ، فقد يكون ذلك بسبب فتح ملف عبر الإنترنت أو عدد من الملفات العالقة في قائمه انتظار أو مزامنة ملف كبيره جدا ، راجع [OneDrive: استكشاف الأخطاء وإصلاحها "معالجه التغييرات"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) للمساعدة في حل هذه المشكلة.

ستري أيضا هذه الرسالة إذا كان لديك ملف يغير كل الوقت (مثل Outlook. ملف PST أو قاعده بيانات). تعرف علي كيفيه [نقل ملفات بيانات Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) التي تستخدمها بشكل منتظم خارج OneDrive.

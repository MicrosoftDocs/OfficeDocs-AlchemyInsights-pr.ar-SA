# <a name="sharepoint-online-calendar-overlay"></a>تراكب تقويم SharePoint Online

يمكنك تراكب تقويم SharePoint آخر طالما أن كلا تقويمي SharePoint في نفس مجموعة الموقع. لا يتم دعم القدرة على استخدام تراكب التقويم في تقويم SharePoint Online أو SharePoint Server الذي يتم استرداده من Exchange Online.

يقدم SharePoint Online تجربة حديثة مع تقويمات مجموعة Microsoft 365. لمعرفة المزيد، راجع [استخدام جزء ويب تقويم المجموعة](https://support.microsoft.com/en-us/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce).

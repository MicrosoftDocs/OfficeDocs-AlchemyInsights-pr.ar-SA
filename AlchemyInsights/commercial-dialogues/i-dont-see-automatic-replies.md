# <a name="i-dont-see-automatic-replies"></a>لا يمكنني رؤية الردود التلقائية

إذا لم تظهر الردود التلقائية بعد تحديد ملف، ربما تستخدم حساب Gmail أو Yahoo أو حساب POP أو IMAP آخر لا يدعم ميزة الردود التلقائية في Outlook. يمكنك إعداد قاعدة للرد على الرسائل الواردة، ولكن فقط إذا تركت Outlook قيد التشغيل.

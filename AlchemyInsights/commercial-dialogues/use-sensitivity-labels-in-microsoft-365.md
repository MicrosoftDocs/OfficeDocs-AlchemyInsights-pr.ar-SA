# <a name="use-sensitivity-labels-in-microsoft-365"></a>استخدام تسميات الحساسية في Microsoft 365

يمكنك استخدام تسميات الحساسية ل:
- فرض إعدادات الحماية مثل التشفير أو العلامات المائية على المحتوى المسمى.
- حماية المحتوى في تطبيقات Office عبر الأنظمة الأساسية والأجهزة المختلفة.
- يمكنك حماية المحتوى في تطبيقات وخدمات الأطراف الخارجية باستخدام Microsoft Cloud App Security.
- توسيع تسميات الحساسية لتشمل تطبيقات وخدمات الأطراف الخارجية.
- صنف المحتوى دون استخدام أي إعدادات حماية.

لمزيد من المعلومات، راجع [التعرف على تسميات الحساسية](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels).

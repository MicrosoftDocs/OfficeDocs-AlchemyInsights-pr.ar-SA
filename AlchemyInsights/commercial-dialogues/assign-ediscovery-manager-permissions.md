# <a name="assign-ediscovery-manager-permissions"></a>تعيين أذونات eDiscovery Manager

لتعيين أذونات eDiscovery Manager، قم بتنفيذ الخطوات التالية:

1. في مركز التوافق [في Office 365 security &،](https://sip.protection.office.com/)حدد أذونات > **eDiscovery Manager**.
2. في الجزء **eDiscovery Manager،** بجانب **eDiscovery Manager،** انقر فوق **تحرير**.
3. في الجزء **تحرير اختيار إدارة eDiscovery،** انقر **فوق اختيار إدارة eDiscovery**.
4. في الجزء **اختيار إدارة eDiscovery،** انقر فوق **+ إضافة**، ثم حدد أسماء المستخدمين.
5. حدد **إضافة > تم > حفظ**.

# <a name="get-started-with-advanced-hunting-queries"></a>بدء استخدام استعلامات البحث المتقدمة

يشارك الباحثون في مجال الأمان من Microsoft [استعلامات البحث المتقدمة بشكل منتظم في GitHub](https://go.microsoft.com/fwlink/?linkid=2144624).

يوفر الباحثون أيضا استعلامات بحث متقدمة حول تحديد موقع الأنشطة والمؤشرات المقترنة [بالتهيات الناشئة](https://go.microsoft.com/fwlink/?linkid=2145808). يتم توفير هذه الاستعلامات كجزء من تقارير تحليل المخاطر، في مركز أمان Microsoft Defender.

# <a name="microsoft-search-setup-guide"></a>دليل إعداد Microsoft Search

استخدم دليل [إعداد Microsoft Search](https://go.microsoft.com/fwlink/?linkid=2153798) لإعداد Microsoft Search وتكوينه. سواء كنت تريد التجربة لمجموعة من المستخدمين أو طرحها للجميع في مؤسستك. سيتم تعيين "مسؤولو البحث" ومحرري "البحث" ثم تخصيص تجربة البحث للمستخدمين باستخدام الإجابات والخيارات الإضافية، مثل إضافة ملحق Bing إلى Chrome أو تعيين Bing كمحرك بحث افتراضي.

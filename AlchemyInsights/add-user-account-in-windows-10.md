# <a name="add-a-user-account-in-windows-10"></a>أضافه حساب مستخدم في Windows 10

اليك كيفيه أضافه حساب في Windows 10:

1. انتقل إلى **قائمه بدء**  >  **Settings**  >  **الحسابات**  >  **& أشخاص آخرين** (أو **أشخاص آخرين**، إذا كنت تستخدم Windows 10 Enterprise) > **أضف شخصا آخر إلى هذا الكمبيوتر الشخصي**.
2. اتبع الإرشادات لأضافه حساب Microsoft أو حساب محلي.

[تعرف علي المزيد حول اعداد الحسابات](https://support.microsoft.com/help/17197/)

# <a name="outlook-search-not-indexing"></a>بحث Outlook لا فهرسة

نحن على علم بمشكلات بحث Outlook المتعلقة بتحديثات الأمان الأخيرة لـ Windows.
  
عند البحث في Outlook، قد تحصل على أحد هذه الأخطاء: **حدث خطأ ما ولم يتم إكمال البحث،** أو قد تكون نتائج البحث غير **مكتملة لأن العناصر لا تزال قيد الفهرسة**. تم إصدار الإصلاح كتحديث Windows اختياري. الرجاء تثبيت واحد المقابلة لإصدار ويندوز كنت ترى هذه المشكلة على: 
  
> [تحديث Windows 7 - 27 يونيو 2017](https://support.microsoft.com/kb/4022168.aspx)
    
> [تحديث Windows 8.1 - 27 يونيو 2017](https://support.microsoft.com/kb/4022720.aspx)
    
> [تحديث Windows 10 الإصدار 1703 - 27 يونيو 2017](https://support.microsoft.com/kb/4022716.aspx)
    
للحصول على أحدث المعلومات، يرجى الاطلاع على المقالة: [مشكلات Outlook المعروفة في تحديثات الأمان يونيو 2017](https://support.office.com/article/Outlook-known-issues-in-the-June-2017-security-updates-3F6DBFFD-8505-492D-B19F-B3B89369ED9B.aspx) المشكلة #5.

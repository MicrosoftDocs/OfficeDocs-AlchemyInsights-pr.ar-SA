# <a name="copy-and-paste-in-outlookcom"></a>النسخ واللصق في Outlook.com

لا توجد أزرار النسخ أو اللصق في Outlook.com ، ولكن يمكنك استخدام اختصارات لوحه المفاتيح:

**علي كمبيوتر شخصي:** اضغط علي CTRL + C للنسخ ، واضغط علي CTRL + X للقص ، واضغط علي CTRL + V للصقه.

**علي جهاز Mac:** CMD + C لنسخها ، و CMD + X للقص ، و CMD + V للصقها.

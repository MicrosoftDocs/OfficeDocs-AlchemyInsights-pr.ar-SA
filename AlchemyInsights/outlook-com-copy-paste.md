# <a name="copy-and-paste-in-outlookcom"></a>نسخ ولصق في Outlook.com

لا يوجد زر نسخ أو لصق في Outlook.com، ولكن يمكنك استخدام اختصارات لوحة المفاتيح:

**على كمبيوتر:** CTRL + C لنسخ، CTRL + X للقص و CTRL + V للصق.

**في نظام التشغيل Mac:** CMD + C لنسخ CMD + X للقص و CMD + V للصق.

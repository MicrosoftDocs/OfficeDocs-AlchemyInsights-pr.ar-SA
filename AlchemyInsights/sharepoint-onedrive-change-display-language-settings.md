# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>تغيير إعدادات لغة العرض في SharePoint أو أندريف 

لا يمكنك تغيير إعدادات اللغة الافتراضية لSharePoint أو أندريف لأنه تم تعريف هذه الإعدادات عند إنشاء اشتراك Office 365. ومع ذلك، يمكنك تغيير إعدادات لغة العرض.

لتغيير إعداداتك الشخصية، راجع [تغيير إعدادات اللغة والمنطقة الشخصية](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

لتغيير واجهة مستخدم الموقع، راجع [اختيار اللغات التي تريد توفيرها لواجهة مستخدم الموقع](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

لمزيد من المعلومات حول استكشاف أخطاء إعدادات اللغة وإصلاحها، راجع لا يمكنك تغيير اللغة الافتراضية [لمجموعة موقع SharePoint على الإنترنت](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection).

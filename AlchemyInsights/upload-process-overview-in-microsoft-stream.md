# <a name="upload-process-overview-in-microsoft-stream"></a>نظرة عامة على عملية التحميل في Microsoft Stream

كلما قمت بتحميل فيديو، يقوم Microsoft Stream تلقائيًا بترميزه في معدلات بت متعددة بحيث يمكن بث الفيديو باستخدام تدفق معدل البت التكيفي. يعمل Microsoft Stream على تحسين معدل البت وجودة الفيديو الذي يتم تشغيله استنادًا إلى عرض النطاق الترددي المتاح للمشاهد.
